Сканирование штрихкода используется для поиска товара и ввода количества в табличной части документов: Заказ, Продажа, Покупка, Анкета, Наличие и цены, Перемещение и Справочнике Товары.

## Алгоритм поиска товара сканированием

Для добавления товара в документ путём сканирования штрихкода выполните следующие шаги:

1.  **Создайте** нужный документ.
2.  **Заполните шапку:** выберите клиента и другие нужные данные.
3.  **Перейдите** на вкладку **«Товары»**. **Больше ничего не выбирайте.**
4.  **Просканируйте** штрихкод товара сканером.

**Если товар найден:** система автоматически выделит соответствующую позицию в списке и увеличит значение в колонке **«Количество»** на 1. При каждом последующем сканировании того же штрихкода значение количества будет увеличиваться на единицу.

**Если товар не найден:** на экране появится сообщение об ошибке: _«**Товар со штрихкодом \[xxxxxxxxxxxx\] не найден**»_.

## **Работа со сканером в режиме эмуляции клавиатуры в Моби-С**

Моби-С поддерживает подключение внешних и встроенных в Терминалы сбора данных (ТСД) сканеров штрихкодов, работающих в режиме эмуляции клавиатуры (HID). Данный режим позволяет использовать практически любые модели сканеров, даже отсутствующие в списке предустановленных.

Для корректной работы необходимо активировать в настройках сканера следующие опции:

> _Примечание: Названия этих настроек могут отличаться в зависимости от модели сканера._

1.  **Использовать внешний сканер:** Включить.
2.  **Режим сканирования:** Выбрать «Клавиатурный ввод» или «Эмуляция клавиатуры».
3.  **Добавлять Enter:** Включить опцию добавления клавиши **Enter** после отсканированного штрихкода.

### Пример настроек для ТСД "Wills Smart"

Запускаем приложение настроек сканера

1.  **Включите сканер:** Активируйте опцию **«Scan turn on»**.
2.  **Выберите режим работы:** В разделе **«Output mode»** установите значение **«Simulate Keyboard»** (Эмуляция клавиатуры).
3.  **Добавьте Enter:** Включите опцию **«Enter key»**, чтобы после сканирования автоматически вводилась клавиша завершения ввода.

## **Сканирование камерой мобильного устройства**

Вариант сканирования штрихкода без использования специализированного сканера. Для сканирования и распознавания штрихкодов используется сторонняя программа **ZXIng сканер**

## Алгоритм поиска товара с помощью камеры мобильного устройства

Для добавления товара в документ путём сканирования штрихкода выполните следующие шаги:

1.  **Создайте** нужный документ.
2.  **Заполните шапку:** выберите клиента и другие нужные данные.
3.  **Перейдите** на вкладку **«Товары»**. **Больше ничего не выбирайте.**
4.  Выберите в меню документа (три точки в верхнем правом углу), пункт **Штрихкод.** Запуститься программа сканирования штрихкодов.
5.  **Наведите** камеру мобильного телефона на штрихкод.

**Если товар найден:** откроется карточка товара и в поле **Кол-во** будет введена цифра **1**.

**Если товар не найден:** на экране появится сообщение об ошибке: _«**Товар со штрихкодом \[xxxxxxxxxxxx\] не найден**»_.

## **Поддержка весовых штрихкодов**

Начиная с версии Моби-С 5.5 (54) реализована поддержка весовых штрихкодов. При сканировании штрихкода происходит его сравнение с **шаблоном**. Весовой штрихкод позволяет искать товар и вводить количество (вес) указанный на штрихкоде. Каждое сканирование добавляет вес к текущему значению.

Шаблон весового штрихкода передается в переменной **ШаблонВесовойШтрихКод** в подзапросе [Settings](/dlya-razrabotchikov/format-obmena-dannyh/FullLoad).

Шаблон это регулярное выражение.

Пример шаблона:(?<Prefix>22)(?<Good>\\d{5})(?<Weight>\\d{2})(?<WeightFract>\\d{3})\\d

Проверка штрихокода по регулярному выражению получает следующие данные:

1.  Prefix - 22 - префикс весового штрихкода
2.  Good - 5 цифр - код товара
3.  Weight - 2 цифры - вес, целая часть
4.  WeightFract - 3 цифры - вес, дробная часть
5.  контрольный символ

## **Дополнительные реквизиты для хранения штрихкода**

Моби-С может не только искать товар по штрихкоду, но и собирать штрихкоды и выгружать их в офис. Для этого используются дополнительные реквизиты. [Дополнительные реквизиты](https://mobi-c.ru/help/mobi_properties.htm). [Настройка дополнительных реквизитов на КПК](https://mobi-c.ru/help/mi8_servicerequisit.htm).

## **Для программиста**

Штрихкод товара выгружается в подзапросе [Units](/funkcii/skaner-shtrihkoda) реквизит **ШтрихКод.** Можно выгружать несколько штрихкодов разделив их запятой.

Редактор форм является визуальным редактором для создания анкетных форм. На мобильном устройстве анкетные формы доступны в документах Анкета. Результатом работы редактора является файл формата XML с описанием содержимого и внешнего вида формы.

Создаваемая анкета может состоять из двух частей. Основная анкета (вкладка Шапка) и анкета привязанная к списку товаров (вкладка Таблица).  Редактор позволяет создавать анкеты с использованием элементов типа Надпись, Поле ввода, Чекбокс, Список и Выпадающий список, Объект и специальный элемент для ввода числового значения. Возможно добавление выгружаемых значений (переменных) и задание [формул](/funkcii/redaktor-form/rabota-s-formulami) для каждого элемента формы.

Для работы редактора форм необходимо наличие на компьютере платформы Microsoft.NET Framework 3.5.

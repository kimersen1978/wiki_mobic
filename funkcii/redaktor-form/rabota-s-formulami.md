Ячейка формы может содержать два типа данных, Текст и формулу. Тип содержимого выбирается в панели инструментов.

![celltype](https://mobi-c.ru/help/celltype.png)

Формула позволят добавлять выражения на встроенном языке с использованием стандартных символов арифметических действий. Аргументами формул могут выступать строки, числа и [переменные](/funkcii/redaktor-form/rabota-s-peremennymi).

Примеры формул:

|     |     |
| --- | --- |
| Формула | Результат |
| <2\*2+33> | 37  |
| <10\*2.58> | 25.80 |
| <Client.Name> | Алхимов А.А. |
| <Client.Address> | 119019, Москва, Арбатская пл. д.25 |

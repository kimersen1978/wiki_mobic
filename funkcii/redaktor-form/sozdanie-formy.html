Редактор форм может быть вызван только из модуля интеграции с 1С. Для создания новой формы нажмите кнопку Создать новую анкету в диалоге Настройка анкет.

Этап 1 Задание имени формы

Имя так же является именем файла в котором хранятся все параметры формы. Например вы ввели название анкеты Мерчендайзинг, редактор создаст файл Мерчендайзинг.xml. Файлы форм хранятся в одной общей папке Forms которая находится в основном каталоге программы Моби-С.

Этап 2 Задание макета формы

Для начала необходимо задать параметры макета меню Макет>Размеры. Диалог Параметры макета задает размер будущей формы. Макет формы представлен в виде таблицы в ячейках которой размещаются элементы  формы. На этом этапе вам необходимо задать количество столбцов и строк будущей анкеты. В будущем вы сможете изменить это значение.

Этап 3 Размещение элементов формы и их содержимого

Для размещения элемента формы выделите мышью ячейку или группу ячеек и выберите на панели Элемент управления требуемый элемент.

Элемент Надпись

Самый простой элемент. Предназначен для отображения текста. Может содержать простой текст, формулу или переменную.

Элемент Галка

Флажок, чекбокс, галочка — элемент графического пользовательского интерфейса, позволяющий пользователю управлять параметром с двумя состояниями — включено и выключено. Возможно задавать вопросы на которые необходимо получить ответы Да или Нет.

Настройки элемента Галка

Верхнее поле ввода позволяет ввести текстовую подпись к флажку. В этом поле возможно использование переменных, кнопка Вставить переменную.

В поле Содержит ответ на вопрос вводится идентификатор вопроса, он необходим для создания типовой анкеты в 1С 8.x и создания документа Опрос при выгрузке заполненной формы из Моби-С.

Если вы установите флажок Обязателен для заполнения то на мобильном устройстве программа не даст сохранить анкету до тех пор пока значение не будет добавлено.

Элемент Выпадающий список

Позволяет пользователю выбрать один элемент из списка. Элементы списка вводятся в верхнее поле ввода и разделяются нажатием клавиши Entere.

Элемент Текстовое поле

Поле для ввода текстовой или числовой информации.

Флажок Многострочный позволяет вводить в поле текстовую информацию в несколько строк

Флажок Числовой позволяет вводить в поле только целые и дробные числа.

Элемент Дата

Добавляет на форму элемент выбора даты.

Элемент Объект

Позволяет выбирать в анкете на КПК значения из базы загруженных из 1С данных (товары, клиенты, фирмы, склады и пр.).

Объекты, такие как Продажа, Заказ, ПКО и другие, берутся из документов, созданных на КПК, которые находятся в Журнале документов. Документы из Архива документов в данном процессе не используются.

Элемент Плюс Минус

Специальный элемент для ввода числовых (целых и дробных) значений. Особенностью данного элемента является то что он запоминает ранее введенные у клиента значения. В новой анкете в элементе открывается предыдущее значение и торговому агенту нужно просто его откорректировать. Идеально подходит для мониторинга цен.

Так элемент Плюс Минус выглядит на мобильном устройстве. Ввод количества производиться свайпом вверх - вниз.

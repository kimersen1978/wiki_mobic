## **Макет**

Позволяет настроить параметры макета. Сколько строк и столбцов будет содержать макет.

![](/funkcii/bar-maket.png)

|     |     |
| --- | --- |
| Элемент | Назначение |
| Объединить/Разъединить ячейки | Объединяет или разъединяет ячейки в выбранной строке. |
| Вставить строку | Добавляет в макет новую строку. |
| Вставить колонку | Добавляет в макет новую колонку. |

Размер макета задается в меню **Макет ⇒ Размеры**. Макет содержит дополнительное меню вызываемое правым кликом мыши.

![](/funkcii/contmenu.png)

## **Ячейка и Элемент управления**

![](/funkcii/bar-element.jpg)

|     |     |
| --- | --- |
| Элемент | Назначение |
| Тип ячейки | Ячейка может содержать обычный текст или [формулу](https://mobi-c.ru/help/formeditorcreateform.htm). |
| Элемент Надпись | Добавить в ячейку элемент Надпись. Отображает заданную текстовую информацию. |
| Элемент Галка | Добавить в ячейку элемент Чекбокс. |
| Элемент Выпадающий список | Добавить в ячейку элемент Список значений. Позволяет выбрать на КПК заранее заданное в списке значение. |
| Элемент Текстовое поле | Добавить в ячейку элемент Поле ввода. Предназначен для ввода информации. |
| Элемент Объект | Позволяет добавить в форму элемент выбора значения из объектов загруженных в КПК. Например, фирма, товар, клиент и пр. |
| Элемент Плюс Минус | Специальный элемент для ввода числовых значений. Запоминает ранее введенное у клиента число. Удобно использовать для мониторинга цен и прочих незначительно изменяющихся числовых значений. |

## **Форматирование**

Панель форматирования позволяет задать шрифт, размер, цвет и прочие параметры отображаемого в ячейке текста.

![](/funkcii/bar-format.png)

|     |     |
| --- | --- |
| Элемент | Назначение |
| Рамка | Создать рамку для ячейки, работает так же с группой ячеек(в новых версиях не работает). |
| Выравнивание | Выравнивание текста относительно ячейки. |
| Шрифт | Выбор типа шрифта. |
| Размер шрифта | Выбор размера шрифта. |
| Жирный | Сделать шрифт жирным. |
| Наклонный | Сделать шрифт наклонным. |
| Подчеркнутый | Сделать шрифт подчеркнутым. |
| Цвет фона ячейки | Выбор цвета фона ячейки |
| Цвет шрифта | Выбор цвета шрифта. |

## **Стандартная**

![](/funkcii/bar-general.png)

|     |     |
| --- | --- |
| Кнопка | Назначение |
| Сохранить | Сохранить текущее состояние формы. |
| Вырезать | Вырезать текст или содержимое ячейки макета в буфер обмена. |
| Копировать | Копировать текст или содержимое ячейки макета в буфер обмена. |
| Вставить | Вставить содержимое буфера обмена. |
| Отменить | Отменить последнее действие. |
| Вернуть | Вернуть предыдущее отмененное действие. |

Для элемента который имеет тип данных формула возможно использование различных переменных. Добавление переменной в анкету позволяет при отображении анкеты на КПК отобразить вместо нее предварительно выгруженное значение. У переменной есть Название и Описание. Название - это уникальное наименование по которому программа определит, что это переменная и по нему будет произведен поиск значения в базе КПК. В коде анкеты используется только название переменной. Описание - это человеко понятное название переменной, используется при выборе переменой из списка.

Пример: (Наименование - Client.Name  Описание - Наименование клиента)

Для того что бы добавить переменную в элемент формы нужно в диалоге Свойства элемента нажать на кнопку Вставить переменную, выбрать из списка.

 

По умолчанию в редакторе есть две предопределенных переменные (Адрес клиента и Наименование клиента). Остальные переменные задают пользователи в учетной системе.

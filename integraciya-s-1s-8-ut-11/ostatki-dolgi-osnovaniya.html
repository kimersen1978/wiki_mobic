Остатки и резерв товара

В настройках агента, на вкладке Склады, задаётся режим расчёта остатков.

Если включён режим Торговля с колес, то остатки по основному складу отображаются с учётом документов, созданных на КПК.

Долг клиента

 

Общий долг клиента или партнера

Для того что бы посмотреть общий долг клиента в вашей 1С, открываем в карточке клиента отчет Задолженность клиентов.

Главное ⇒ Партнеры ⇒ Карточка партнера ⇒ Отчеты ⇒ Задолженность клиентов.

  • Столбец Долг клиента - задолженность клиента.
  • Столбец Наш долг - наша задолженность перед клиентом.
  • Столбец Просрочено - сумма просроченного долга. Клиенты с просроченным долгом выделяется на КПК красным цветом. Настройка Запретить выбирать клиентов с просроченным долгом позволяет запретить оформление документов Заказ и Продажа.

Внимание! В КПК долг клиента выгружается общий, без учета разделения по фирмам.

Внимание! Долги в отчете отображаются в валюте документа. На КПК долг пересчитывается в валюту заданную в Настройки агента ⇒ вкладка Настройки > настройка Валюта на КПК. Курс пересчета находится в Регистр сведений > Курсы валют.

Документы основания для ПКО и архива документов

Долг по документу выгружается на момент выполнения полной загрузки/обновления данных, если по документу была частичная оплата, сумма долга по документу будет с учетом частичной оплаты.

Главное ⇒ Партнеры ⇒ Карточка партнера ⇒ Отчеты ⇒ Задолженность клиентов.

 

Параметры влияющие на выгрузку документов оснований

1. Выгружаются только документы Заказ и Реализация.

2. Настройки агента ⇒ вкладка Настройки ⇒ кнопка ПКО, отроется диалог Настройки документа ПКО

  • Список выбора Создавать, значение "Приходный кассовый ордер", если выбрано значение “Запрещен для ввода с КПК”, то документы основания не выгружаются.
  • Список выбора Период -  задает период выборки документов.
  • Список выбора Форма оплаты  - фильтр выгружаемых документов по типу оплаты.
  • Флажок Только документы агента  - только документы в которых на вкладке Дополнительно в поле Менеджер выбран текущий торговый агент.

3. Настройки агента ⇒ вкладка Склады - только документы на склады из списка.

4. Настройки агента ⇒ вкладка Организации - только документы на организации из списка.

5. Настройки агента ⇒ вкладка Типы цен - только документы на типы цен из списка.

6. Соглашение из документа должно быть доступно агенту. Список доступных соглашений можно посмотреть в отчете Настройки агента ⇒ вкладка Контрагенты кнопка Клиенты и соглашения агента.

Если по всем этим параметрам документ должен быть доступен необходимо проверить корректность заполнения клиента, организации, договора/соглашения и не помечен ли на удаление договор/соглашение.

Цены

Для того что бы посмотреть текущие цену по товару в вашей 1С, открываем в карточке товара функцию Прайс-лист компании.

Выбор цены на КПК

Почему цена на КПК может отличаться от цены в прайс листе?

На цену товара влияет галка Цена включает НДС в настройках цены. Перейдите в справочник Виды цен и откройте настройки нужной цены.

  • Флажок Цена включает НДС установлен - на КПК отображается цена из прайс листа. Смотрите цену Дилерская на верхних скриншотах.
  • Флажок Цена включает НДС снят - на КПК отображается цена из прайс листа + НДС. Процент НДС можно посмотреть в карточке товара. Смотрите цену Дистрибьюторская на верхних скриншотах, добавился НДС 18%.

Если для товара не установлена цена по выбранному типу цен, то на КПК подставиться цена “по умолчанию”. Настройки агента ⇒ вкладка Цены и скидки ⇒ параметр Тип цены по умолчанию.

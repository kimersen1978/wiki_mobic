Диалог Состав печатных форм на КПК позволяет сформировать список и название печатных форм, загружаемых в мобильное устройство агента.

  • Каждая печатная форма - это отдельный файл в формате xml.
  • Файлы печатных форм хранятся в каталоге "Forms" который находится в Основном каталоге Моби-С.
  • Если каталог "Forms" пустой скачайте файлы печатных форм с нашего сайта и положите в этот каталог.
  • Печатные формы можно редактировать и создавать новые.
  • Как настроить печать и подключить принтер на КПК?
  • Печать работает только для лицензий Моби-С Pro.
  • Рекомендации по автоматизации торговли с колёс.

На КПК торговых агентов выгружаются печатные формы, для которых установлен флажок в поле Исп. В столбце Наименование задаётся отображаемое в КПК название формы. В столбце Файл формы(XML) выбирается путь к файлу печатной формы. В столбце Владелец выбирается документа для которого будет доступна эта печатная форма.

В верхнем тулбаре находятся кнопки Добавления и Удаления печатных форм из списка и кнопки сортировки и тонкой сортировки печатных форм. Порядок печатных форм на КПК будет такой же, как в этом списке.

Печатные формы загружаются на КПК торгового агента при выполнении запроса Полная загрузка.

Совет! Если торговые агенты не используют в работе печать документов, рекомендуем снять все галки в поле "Исп", это снизит обьем передаваемых на КПК данных.

Для каждого документа на КПК есть функция "Печать". При использовании открывается список подключенных печатных форм. Для каждой печатной формы можно выбрать количество экземпляров. Выбранные формы  можно распечатать на принтере или отправить в pdf формате по электронной почте.

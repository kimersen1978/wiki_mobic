Все операции с лицензией (добавление, смена агента) выполняйте только при одной запущенной обработке «Моби-С». Запуск нескольких обработок одновременно может заблокировать лицензию и вызвать ошибку: «Ошибка активации: Нельзя активировать один и тот же ключ более 5 раз в сутки». Подробнее см. «Правило №3».

Открывает диалог Лицензионные ключи на КПК.  Диалог предназначен для ввода и хранения лицензионных ключей Моби-С.

Подробную информацию по лицензированию можно получить на нашем сайте в разделе Лицензирование.

Кнопка Загрузить ключи позволяет сделать массовую загрузку лицензионных ключей из текстового файла. Текстовый файл с ключами можно взять в личном кабинете на нашем сайте или из файла сделанного через кнопку Сохранить в файл.

Кнопка Сохранить ключи в файл позволяет сделать резервную копию лицензионных ключей в текстовый файл.

Вкладка Лицензии с привязкой к оборудованию

В данной вкладке хранятся лицензии с привязкой к оборудованию. Для добавления нового ключа нажмите в верхнем тулбаре кнопку Добавить. После ввода кода КПК и ключа в столбце Версия будет выведен статус ключа Профессиональная или Стандартная. Если в поле будет надпись Ошибка значит вы неправильно ввели лицензионные данные или они неверны. В столбец Комментарий можно ввести любую информацию, имя торгового агента или устройство, для которого приобретён лицензионный ключ и пр.

Лицензии с привязкой к оборудованию устаревшие, приобретайте их только, если ваша версия Моби-С не поддерживает лицензии с онлайн активацией.

Вкладка Лицензии с онлайн активацией

В данной вкладке хранятся лицензии с онлайн активацией.

  • Для добавления нового ключа нажмите в верхнем тулбаре кнопку Добавить.
  • В поле Лицензионный ключ добавьте лицензию.
  • Поле Агент, для того, чтобы задействовать лицензию её необходимо привязать к существующему торговому агенту.
  • Поле Версия выводится тип лицензии, Профессиональная или Стандартная.
  • Поле Последний обмен, дата и время последнего запроса данных, пришедшего с КПК. Позволяет понять используется лицензия или её можно назначить другому агенту.
  • Поле Статус - Свободна, лицензия никогда не активировалась, Используется, лицензия был активирована, Заблокирована, лицензия в блоке.

После любых изменений нажмите кнопку Применить, чтобы сохранить список лицензий и получить их текущий статус.

Внимание! Для работы лицензий необходимо пробросить порт 443 HTTPS до сайта mobi-c.ru.

Установка и настройка сервиса Моби-С:Заказы

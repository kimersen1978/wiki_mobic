Информация актуальна для конфигураций построенных на базе Управление торговлей 11.

Запуск Моби-С в режиме обмена с КПК рекомендуется выполнять от имени пользователя с полными правами. В случае если предоставить пользователю полные права невозможно, тогда их можно ограничить с учетом выполнения следующих требований:

Если указан доступ на чтение к элементу Справочник.Справочник1, то доступ на чтение должен быть и ко всем реквизитам данного справочника.

Обязательные права для запуска обработки:

  • у пользователя должно быть право на запуск (внешних) обработок
  • у пользователя должно быть право Администрирование (при запуске файл-открыть) или обработка должна быть встроенной в конфигурацию или добавлена в расширение
  • у пользователя должно быть право на чтение/запись/изменение для объекта Справочники.ГруппыПользователей, Справочники.Пользователи
  • у пользователя должно быть право на чтение/запись/изменение для объекта ПланВидовХарактеристик.ДополнительныеРеквизитыИСведения, Справочники.ЗначенияСвойствОбъектов
  • у пользователя должно быть право на чтение/запись/изменение для РегистрСведений.ДополнительныеСведения
  • у пользователя должно быть право на чтение для объектов Справочники.Номенклатура, Справочник.ХарактеристикиНоменклатуры, Справочник.СерииНоменклатуры
  • у пользователя должно быть право на чтение для объекта Справочники.Контрагенты, Справочники.Партнеры.
  • у пользователя должно быть право на чтение/запись/изменение для объекта Справочники.ВидыКонтактнойИнформации
  • у пользователя должно быть право на чтение для объекта Справочники.СегментыНоменклатуры
  • у пользователя должно быть право на чтение для объекта Справочники.СегментыПартнеров
  • у пользователя должно быть право на чтение для объекта РегистрСведений.НоменклатураСегмента
  • у пользователя должно быть право на чтение для объекта РегистрСведений.ПартнерыСегмента

Для редактирования настроек агентов необходимо предоставить право на доступ на чтение для следующих объектов:

  • Справочники.Валюты
  • Справочники.СтатьиДвиженияДенежныхСредств
  • Справочники.СтруктураПредприятия (Справочники.Подразделения)
  • Справочники.ВидыЦен
  • Справочники.СкидкиНаценки
  • Справочники.Склады
  • Справочники.Организации
  • Справочники.Кассы
  • Справочники.КассыККМ

Для назначения форматов торговых точек предоставить право на чтение/запись/изменение:

  • РегистрСведений.ДополнительныеСведения
  • Документ.ИзменениеАссортимента
  • Справочники.ФорматыМагазинов

Для назначения заданий агентам необходимо предоставить право на чтение/запись/изменение/удаление:

  • Документ.ЗаданиеТорговомуПредставителю
  • Справочник.ПапкиФайлов
  • Справочник.Файлы

Для запуска Моби-С в режиме обмена с КПК необходимо предоставить доступ на чтение:

  • РегистрСведений.АналитикаУчетаПоПартнерам чтение и запись
  • РегистрНакопления.СвободныеОстатки
  • РегистрНакопления.РасчетыСКлиентами
  • РегистрНакопления.РасчетыСПоставщиками
  • РегистрыСведений.ДополнительныеСведения
  • Справочник.ПартнерыПрисоединенныеФайлы
  • Справочник.НоменклатураПрисоединенныеФайлы
  • РегистрСведений.ПрисоединенныеФайлы
  • Справочник.СоглашенияСПоставщиками
  • Справочник.СоглашенияСКлиентами
  • РегистрыСведений.КурсыВалют
  • Справочник.КонтактныеЛицаПартнеров
  • Справочник.ДоговорыКонтрагентов
  • Справочник.УпаковкиЕдиницыИзмерения (Справочник.Упаковки, Справочник.ЕдиницыИзмерения)

Для выгрузки фотографий из 1С в КПК необходимо дать права на чтение следующих объектов

  • Справочник.НоменклатураПрисоединенныеФайлы для выгрузки фотографий номенклатуры
  • Справочник.ПартнерыПрисоединенныеФайлы для выгрузки фотографий клиентов
  • Справочник.Файлы для выгрузки фотографий заданий
  • при использовании хранения фотографий в томах - необходимо дать право на доступ на чтение к Справочник.ТомаХраненияФайлов

доступ на чтение/изменение/запись/проведение:

  • к документам разрешенным агенту на создание, запись и проведение
  • к Справочники.Контрагенты, Справочники.Партнеры в случае если агент имеет право регистрировать новых клиентов.
  • Справочники.КлючиАналитикиУчетаПоПартнерам

Для работы с анкетами (в том числе и по товарам) необходимо предоставить агенту право работы с этими объектами.

Если в Вашей конфигурации используется ограничение доступа на уровне данных (RLS) то в этом случае агент будет видеть только разрешенные (не запрещенные) ему объекты.

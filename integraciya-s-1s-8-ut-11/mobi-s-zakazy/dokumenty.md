# Статусы

![](/nhelp7.png)

В Моби-С.Заказы доступны 7 статусов.

Поле **Наименование** \- редактируемое название статуса.

Флажок **Заказ завершен** - документ с этим статусом будет отображаться на КПК в разделе **Завершенные**.

Флажок **Архивный** \-  скрывает статус из доступных для выбора в диалоге **Документы**.

Стрелки вверх вниз позволяют задать порядок отображения статусов заказа в диалоге **Выберите статус**.

После редактирования параметров статуса нажмите кнопку **Записать**.

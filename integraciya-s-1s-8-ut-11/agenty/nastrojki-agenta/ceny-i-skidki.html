На вкладке Цены и скидки типы цен и скидок, которые будут выгружаться на КПК.

Поле Тип цен по умолчанию (обязательно к заполнению), элемент справочника Типы цен номенклатуры. Цена "по умолчанию" используется на КПК тогда, когда информация о типе цен недоступна, например, когда не выбран клиент.

Блок Типы цен

Настройка списка список типов цен, которые будут выгружаться на КПК. Флажок Типы цен из соглашений, Моби-С выгружает на КПК только типы цен, которые встречаются в выгружаемых соглашениях контрагентов. Ниже, в таблице, перечислен список типов цен зарегистрированных в системе. Если флажок Типы цен из соглашений снят, то на КПК будут выгружены только отмеченные галкой типы цен.

Блок Типы скидок

Настройка списка скидок, которые будут выгружаться на КПК. Эти скидки используются на КПК в карточке товара. Флажок Скидки из соглашений, Моби-С выгружает на КПК только скидки, которые встречаются в выгружаемых соглашениях контрагентов. Ниже в таблице перечислен список скидок зарегистрированных в системе. Если флажок Скидки из соглашений снят, то на КПК будут выгружены только отмеченные галкой скидки. Для добавления новой строки скидок в справочнике Скидки (наценки) создайте Скидку (наценку) процентом, задайте название и процент скидки, параметра Назначается установите Вручную, скидка должна применятся без каких либо условий.

Моби-С поддерживает персональные цены контрагентов заданных в Соглашениях на вкладках Уточнение цен по товарам и Уточнение по ценовым группам. Флажок Выгружать уточнения цен из соглашений, генерирует для каждого клиента свой тип цены с учётом Уточнений по товарам и ценовым группам.

Внимание! Установка этого флажка генерирует большой объём данных. Запрос Полная загрузка может выполняться продолжительное время.

Для работы  с персональными ценами и скидками контрагента можно использовать на КПК функции Коррекция и Цены контрагента.

Расчёт наценки

Функция Расчёт наценки выгружает на КПК, для всех товаров цену, которая считается закупочной. Зная закупочную и продажную цену можно оценить доходность продажи товара или всего документа.

  • Не доступно на КПК - функция Закупочная цена не используется.
  • Равна типу цены - выбрать конкретный тип цен из справочника Типы цен номенклатуры.
  • [Тип цены] - последнее выбранное значение из справочника Типы цен номенклатуры.

Флажок Запретить изменять цены и скидки на КПК, установлен то запрещено редактирование цен и скидок на КПК для..

  • Для всех - для всех клиентов
  • для партнёров из сегмента - только для клиентов которые входят в выбранный сегмент

Если флажок сброшен, то разрешено редактирование для всех клиентов.

Настройка действует для документов Заказ, Продажа, Покупка.

Флажки Разрешать изменять цену, тип цены, скидку, разрешают редактировать выбранные данные в карточке товара на КПК для..

  • Для всех - для всех товаров
  • для товаров из сегмента - только для товаров которые входят в выбранный сегмент

По умолчанию все флажки сброшены и редактирование цены и скидки в карточке товара запрещены.

Настройка действует для документов Заказ и Продажа.

 
Внимание! Настройка Запретить изменят цены и скидки на КПК имеет приоритет.

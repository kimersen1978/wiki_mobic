## **Справочник Торговые агенты**

Список отображает только доступных для текущего пользователя 1С агентов. Каким пользователям доступны агенты, задается на вкладке **Супервайзеры** в настройках агента.

Пользователю с правами **Администратор** доступны все агенты.

Торговый агент это пользователь 1С который входит в группу пользователей "**Mobi-C:Агенты**". Проверьте у пользователя наличие **Физического лица**. Кодом агента является имя агента в справочнике. Укажите **Код агента** в настройках базы данных на мобильном устройстве. Для дополнительной защиты обмена данными от несанкционированного доступа, рекомендуется вручную проставить сложные коды агента, чтобы исключить доступ к данным методом перебора. Доступ к подключению к базе разрешён только для агентов из данного списка.

Внимание! Пользователь **Mobi-C:(служебный)** создается обработкой Моби-С и предназначен для хранения глобальных настроек. Использовать его в качестве агента или для входа в 1С нельзя.

![](/ut11/ut11-01.png)

В таблице отображается **Имя агента** одновременно являющееся и **Кодом агента**, а так же дата и время последнего обмена агента с 1С.

Для создания нового торгового агента нажмите кнопку **Создать**. Настройки нового агента заполняются из настроек агента который последний выполнял "**Полную загрузку**".

Для редактирования настроек агента дважды кликните на нем мышью. Откроется диалог [**Настройки**](https://wiki.mobi-c.ru/ru/integraciya-s-1s-8-ut-11/agenty/nastrojki-agenta).

Для удаления агента из списка нажмите кнопку **Удалить** (красный крестик). Кнопка **Удалить,** удаляет агента только из списка агентов Моби-С.

Совет: Задайте "оптимальные настройки" для одного агента. Воспользуйтесь кнопкой **Скопировать** (маленький плюсик), что бы быстро создать новых агентов с оптимальными настройками. Выделите строку агента у которого копируем настройки. Нажимаем кнопку **Скопировать**.

Кнопка **Групповая обработка** предназначена для быстрой настройки большого количества торговых агентов и позволяет копировать и редактировать настройки агента.

![](/ut11/ut11-11.png)

1.  В поле **Загрузить настройки из** выберите торгового агента, у которого будем копировать настройки.
2.  Нажмите кнопку **Редактировать** в верхнем тулбаре, чтобы скорректировать копируемые настройки.
3.  В разделе **Применить настройки к следующим агентам** сформируйте список агентов.
4.  Нажмите кнопку **Скопировать настройки**, чтобы скопировать настройки.

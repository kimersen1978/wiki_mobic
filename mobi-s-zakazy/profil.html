Раздел Профиль содержит данные по поставщику, функцию обновления данных и информацию о клиенте.

Для начала работы с Моби-С.Заказы необходимо загрузить данные поставщика. В разделе Магазин можно выбрать текущего актуального поставщика, удалить, добавить нового и обновить данные о товарах. Поставщиков может быть несколько.

В разделе Заказы, находятся текущие и ранее оформленные заказы.

Можно открыть и просмотреть содержимое любого ранее оформленного заказа. Содержимое раздела Заказы задается в настройках Моби-С.Заказы в 1С, кнопка Настройка выгрузки на КПК, вкладка Архив документов. Для того чтобы оформить заказ повторно, откройте заказ и нажмите на кнопку с корзиной в верхнем правом углу. В Корзину добавится все содержимое заказа.

Для отправки оформленных в "Корзине" заказов обязательно нужно заполнить все данные в разделе Личные данные. Раздел Мобильный телефон требует проверки кодом, который будет отравлен по SMS.

Добавление нового магазина

Для добавления перейдите в раздел Магазин и нажмите кнопку с плюсиком (в правом нижнем углу).

Отобразится список опубликованных в системе магазинов, к которым может подключиться любой желающий. Для того что бы ваш магазин попал в это список обратитесь в нашу техническую поддержку.

Если вашего магазина нет в списке, то для добавления нажмите кнопку с QR-кодом (в правом нижнем углу). Добавить поставщика можно двумя способами: отсканировать QR код или ввести строку настроек. Эту информацию вам должен предоставить сам магазин.

1.Скопируйте строку настроек в буфер обмена. Вставьте её в поле Строка настроек и нажмите кнопку ОК.

2.Нажмите кнопку QR-КОД. Запустится программа сканирования с помощью встроенной камеры. Отсканируйте QR-код.

Возможные ошибки

Вам необходимо зарегистрироваться у поставщика - телефонный номер клиента не зарегистрирован в 1С. Добавьте номер телефона в настройках клиента на форме Моби-С.Заказы.

Сервер поставщика не отвечает (ERROR_NO_ACC_SYSTEM) - не загружены данные на сервер. После настройки данных в Моби-С.Заказы в 1С необходимо нажать кнопку Выгрузить данные.

Unable to resolve host "sky.mobi-c.ru": No address associated with hostname - на КПК отсутствует интернет.

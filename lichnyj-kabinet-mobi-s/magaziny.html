Раздел "Магазины" содержит настройки для работы с сервисом Моби-С.Заказы.

Добавление нового магазина

Для добавления нового магазина нажмите в правом нижнем углу кнопку с плюсом.

  • Наименование - наименование магазина.
  • Логотип - логотип магазина.
  • Адрес или регион магазина -
  • Телефоны -
  • Сайт -
  • Код учетной системы и Пароль учетной системы - возьмите эти данные в вашей обработке Моби-С в настройках Mobi-C.Net.
  • Описание - расширенное описание вашего магазина.

После создания магазина ему будет присвоен идентификатор, который потребуется при настройке Моби-С.Заказы в 1С.

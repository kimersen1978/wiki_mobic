Добавление на главный экран программы кнопки и пункт меню Справка.

В справочной информации можно выводить любую информацию необходимую для работы торгового агента

  • Справку по работе с программой
  • Должностные инструкции
  • Изображения планограммы или выкладки товара

Справка может содержать оглавление и любое количество страниц. Справка пишется в формате HTML, поддерживается все теги и загрузка изображений.

Подготовка справки

Скачать пример файла help.zip. В примере простейшая одностраничная справка.

Для того, чтобы сделать справку из нескольких страниц с оглавлением, необходимо использовать специальную обработку СборкаСправочнойИнформации 8.2.epf

Скачать обработку для платформы 1С Предприятие 8.2

Требования к справке

  1. Каждый раздел справки должен располагаться в отдельном каталоге.
  2. Каждый раздел должен содержать файл index.html.
  3. Если в разделе используются изображения, то они должны находиться в каталоге файла index.html (допускаются подкаталоги). Ссылки на изображение в файле index.html должны быть ОТНОСИТЕЛЬНЫМИ.

Загрузка справки на КПК

1.Положите файл help.zip в Основной каталог Моби-С\Updates\Android\

2.В этом же каталоге откройте для редактирования файл update.ini, измените содержимое на.

1000
help.zip

3.Запустите Моби-С на мобильном устройстве и выполните запрос Полная загрузка. В начале запроса будет скачан на КПК и установлен файл help.zip.

Если все сделали правильно, то на рабочем столе Моби-С появится кнопка Справка.

Внимание! После того как все агенты сделают Полную загрузку, очистите содержимое файла update.ini. Если это не сделать, файл help.zip будет загружаться при каждом выполнении запроса.

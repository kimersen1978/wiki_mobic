Добавление на главный экран программы кнопки которая выводит диалог с произвольной текстовой информацией. По умолчанию в нем размещена информация о технической поддержке Моби-С. Партнеры могут разместить в этом диалоге информацию о своей компании и указать свои координаты техподдержки.

Для кнопки можно изменить подпись и иконку. В диалоге возможно использование своего логотипа и информации в формате html (форматирование, гиперссылки и пр.).

1.Скачайте файл [support.zip](https://mobi-c.ru/arc/support.zip)

> Содержимое файла support.zip
> 
> Папка icon
> 
> файлы пиктограммы кнопки в различных разрешениях
> 
> файл label.txt - содержит строку с подписью к кнопке
> 
> Папка page
> 
> файл logo.png - изображение логотипа  
> файл page.html - информация

2.Распакуйте файл **support.zip**, измените требуемые файлы, запакуйте обратно. Обратите внимание, в архиве support.zip должны быть только папки **icon** и **page**.

3.Положите файл **support.zip** в Основной каталог Моби-С\\Updates\\Android\\

4.В этом же каталоге откройте для редактирования файл update.ini, измените содержимое на.

`1000`  
`support.zip`

5.Запустите Моби-С на мобильном устройстве и выполните запрос "**Полная загрузка**". В начале запроса будет скачан на КПК и установлен файл **help.zip**.

Если все сделали правильно, то на рабочем столе Моби-С появится кнопка "Техподдержка".

![](/mobilnyj-klient/screenshot_2013-07-03-15-18-56.png)![](/mobilnyj-klient/screenshot_2013-07-03-15-19-07.png)

Внимание! После того как все агенты сделают **Полную загрузку,** очистите содержимое файла update.ini. Если это не сделать, файл **support.zip** будет загружаться при каждом выполнении запроса.

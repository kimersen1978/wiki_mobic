# **ErrorLog – запрос на загрузку отчёта об ошибках**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
ErrorLog\r\n
[<ДатаВремя>\t<Порядок>\t<Важность>\t<СтрокаОтчетаОбОшибках>\r\n]
Ответ:
OK
\r\n
```

Назначение: На КПК ведется лог ошибок работы Моби-С. Запрос выгружает этот лог с КПК в учетную систему.

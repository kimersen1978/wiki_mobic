#  **TaskReport – запрос отчета по выполненным заданиям**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
TaskReport\r\n
<ДатаВремя>\t<КодДок>\t<НомерДок>\t<КодСобытия>\t<Статус>\t<Отчет>\t<Фото>\t<ДатаВремяСохраненияДокумента>\t<ВремяРаботы>\t<ДатаВремяСозданияДокумента>\t<СозданиеШирота>\t<СозданиеДолгота>\t<ДатаВремяВыгрузкаДокумента>\t<ВыгрузкаШирота>\t<ВыгрузкаДолгота>\t<ИдентификаторДокумента>\r\n
\r\n
Ответ:
OK\r\n
[<КодДок>\t<?>\t<СообщениеОбОшибке>\r\n]
\r\n
 
```

| Параметр | Назначение |
| --- | --- |
| ДатаВремя | Дата и время выполнения задания в КПК. |
| КодДок | Номер документа созданного в результате выполнения задания. |
| НомерДок | Формула формирования номера документа<br><br>Префикс+Месяц+День+Номер документа<br><br>Префикс - последние 4 знака кода агента.<br><br>Месяц - число 2 знака, месяц создания документа<br><br>День - число 2 знака, день месяца создания документа<br><br>Номер документа - число 2 знака, номер документа в КПК |
| КодСобытия | Код документа Событие в учетной системе. |
| Статус | Текущий статус задания. 1 - Завершено, другое значение - Отменено. |
| Отчет | Текстовое описание результатов выполнения задания. Добавляется в поле Комментарий документа Событие. |
| Фото | Добавляет в поле Комментарий документа _Событие_ имя фото. |
| ДатаВремяСохраненияДокумента | Дата и время первого сохранения документа на КПК. Может быть использовано для расчета времени затраченного на создание документа. |
| ВремяРаботы | Время работы над заданием в минутах |
| ДатаВремяСозданияДокумента | Дата и время создания документа. Берется с GPS приемника. Выгружается в UTC формате (часовой пояс +0). Если координаты нет это значит, что нет GPS приема или точность координаты более 200 метров. |
| СозданиеШирота | GPS широта, в формате градусы с десятичной дробной частью |
| СозданиеДолгота | GPS долгота, в формате градусы с десятичной дробной частью |
| ДатаВремяВыгрузкаДокумента | Дата и время выгрузки документа.  Берется с GPS приемника. Выгружается в UTC формате (часовой пояс +0). Если координаты нет это значит, что нет GPS приема или точность координаты более 200 метров. |
| ВыгрузкаШирота | GPS широта, в формате градусы с десятичной дробной частью |
| ВыгрузкаДолгота | GPS долгота, в формате градусы с десятичной дробной частью |

 Создание и контроль выполнения заданий торговым агентом для стандартных конфигураций 1С 8.1 и 8.2 реализовано на базе документа **Событие**.

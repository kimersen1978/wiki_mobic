# **FileLoad – запрос файла**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
FileLoad\r\n
<ИмяФайла>\r\n
Ответ:
OK\r\n
<Двоичные данные, формируемые внешней компонентой PocketServer.dll, методом ПослатьФайл>
\r\n
```

Двоичные данные представляют собой строку с разделителем строк – размер файла в байтах, далее идёт сжатый по алгоритму ZIP двоичный поток. От обработки требуется только передать **PocketServer.dl**l путь к файлу, остальное внешний отчет сделает сам.

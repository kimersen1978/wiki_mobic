# **FastRests – запрос на  быстрое обновление остатков в КПК**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
FastRests\r\n
[Параметры запроса отсутствуют]
Ответ:
OK\r\n
[<КодТовара> \t<КодСклада> \t<Остаток> \t<КодФирмы>\t<Отображать>\r\n]
\r\n
```

| Параметр | Назначение |
| --- | --- |
| КодТовара | Код товара из справочника товаров. |
| КодСклада | Код склада |
| Остаток | Остаток товара. |
| КодФирмы | Код организации. |
| Отображать | Цифровой код который задает вариант отображения остатка на КПК.<br><br>0 - остаток на КПК отображается без ограничений<br><br>1 - максимальный остаток ограничен в настройках<br><br>2 - остаток на КПК скрыт, по умолчанию применяется для услуг, наборов, комплектов |

Запрос FastRest выполняется сразу же после выгрузки документов с КПК.

GoodRests\\r\\n \[<КодТовара> \\t<КодСклада> \\t<Остаток> \\t<КодФирмы>\\t<Отображать>\\r\\n\]

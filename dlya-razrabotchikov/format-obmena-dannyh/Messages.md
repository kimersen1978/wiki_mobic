# **Messages – запрос загрузку сообщений**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
Messages\r\n
[Параметры запроса отсутствуют]
Ответ:
OK\r\n
[<КодСообщения> \t<ДатаВремяСоздания> \t<Автор> \t<КодТаблицы> \t<КодОбьектаТаблицы> \t<Тема> \t<Сообщение> \t<ДатаАктуальности> \t<Статус> \t<ОбязательноКВыполению>\r\n]
\r\n
```

| Параметр | Назначение |
| --- | --- |
| ∗КодСообщения | Код сообщения в учетной системе. |
| ДатаВремяСоздания | Дата и время создания сообщения. |
| Автор | Автор сообщения. Сейчас в КПК не задействовано. |
| КодТаблицы | Внутренний номер таблицы на КПК. 1-Контрагенты, 4-Номенклатура, 6-Агенты |
| КодОбьектаТаблицы | Кому адресовано сообщение. Код номенклатуры, клиента или агента. |
| Тема | Тема сообщения. Сейчас в КПК не задействовано. |
| Сообщение | Текст сообщения. Строки можно разделять символом VTab. |
| ДатаАктуальности | Дата до которой сообщение актуально. Сейчас в КПК не задействовано. |
| Статус | Статус сообщения. 1-Новое,  2-Отправлено, 3-Прочитано, 4-Выполнено |
| ОбязательноКВыполению | Флаг важности. Сейчас в КПК не задействовано. |

Сообщения в КПК выгружаются когда торговый агент делает один из трех запросов(Полная загрузка, Обновить остатки, Выгрузка документов).

# **Route – запрос на расчет и выгрузку маршрута движения между двумя клиентами**

> Запрос устарел, больше не поддерживается.

`<ВерсияПротоколаОбмена>\r\n`  
`<Платформа>\r\n`  
`<КодАгента>\r\n`  
`Route\r\n`  
`<Широта>\t<Долгота>\r\n`  
`<КодКлиентаОткуда>\t<КодКонтрагентаКуда>\r\n`

`Ответ:`  
`ОК\r\n`  
`<ШиротаНач>\t<ДолготаНач>\t<ШиротаКон>\t<ДолготаКон>\r\n`  
`[<ШиротаТочки>\t<ДолготаТочки>\r\n]`  
`\r\n`

Расчет маршрута между двумя клиентами и выгрузка его в КПК.

| Параметр | Тип в КПК | Назначение |
| --- | --- | --- |
| Широта | строка | Значение широты GPS координаты текущего местонахождения агента. |
| Долгота | строка | Значение долготы GPS координаты текущего местонахождения агента. |
| КодКлиентаОткуда | строка | Код клиента от которого будем прокладывать маршрут. |
| КодКонтрагентаКуда | строка | Код клиента к которому будем прокладывать маршрут. |
|     |     |     |
| ШиротаНач | строка | Значение широты GPS координаты начальной точки маршрута. |
| ДолготаНач | строка | Значение долготы GPS координаты начальной точки маршрута. |
| ШиротаКон | строка | Значение широты GPS координаты конечной точки маршрута. |
| ДолготаКон | строка | Значение долготы GPS координаты конечной точки маршрута. |
| ШиротаТочки | строка | Значение долготы GPS координаты промежуточной точки маршрута. |
| ДолготаТочки | строка | Значение долготы GPS координаты промежуточной точки маршрута. |

# **SalesHelp – запрос на загрузку истории и плана продаж**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
SalesHelp\r\n
[<КодКлиента>\r\n]
Ответ:
OK\r\n
[<КодКлиента>\t<КодТовара>\t<ПланПродаж>\t<ИсторияПродаж>\r\n]
```

Назначение: Выгружает дополнительную информацию о товарах для заданного клиента. История продаж и план продаж.

|     |     |
| --- | --- |
| Параметр | Назначение |
| ∗КодКлиента | Код клиента в учетной системе. |
| ∗КодТовара | Код товара в учетной системе. |
| План | План продаж. |
| ИсторияПродаж | История продаж. |

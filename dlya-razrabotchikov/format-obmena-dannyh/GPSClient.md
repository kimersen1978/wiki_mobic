# **GPSClient – запрос на выгрузку GPS координат клиентов**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
GPSClient\r\n
[<КодКлиента>\t<Широта>\t<Долгота>\r\n]
Ответ:
OK
\r\n
```

| Параметр | Назначение |
| --- | --- |
| КодКлиента | Код клиента в учетной системе. |
| Широта | Широта в формате градусы с десятичными долями. |
| Долгота | Долгота в формате градусы с десятичными долями. |

Выгрузка GPS данных происходит в момент выполнения запроса [**Выгрузка документов**](/mobilnyj-klient-na-android/obmen-dannymi/vygruzka-dokumentov).

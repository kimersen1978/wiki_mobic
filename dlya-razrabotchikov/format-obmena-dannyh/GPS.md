# GPS - запрос на выгрузку данных GPS трека

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
GPS\r\n
[<ДатаВремя>\t<Долгота>\t<Широта>\t<Точность>\t<Скорость>\t<Индекс>\t<ТипКоординаты>\r\n]
Ответ:
OK
\r\n
```

| Параметр | Назначение |
| --- | --- |
| ДатаВремя | Дата и время снятия координаты в универсальном времени(UTC).  <br>Требует дополнительной конвертации в локальное время. |
| Долгота | Долгота в формате градусы с десятичными долями. |
| Широта | Широта в формате градусы с десятичными долями. |
| Точность | Точность GPS координаты. |
| Скорость | Скорость движения (километры в час). |
| Индекс | Уникальный идентификатор строки. |
| ТипКоординаты | 0 — Fake, 1 — GPS, 2 — внешний приёмник GPS, 3 — Network (Wi-Fi), 4 — неизвестный провайдер |

GPS трек выгружается из КПК в 1С двумя способами

1.  Автоматически каждые 15 минут.
2.  При выполнении запрос [Выгрузить документы](/mobilnyj-klient-na-android/obmen-dannymi/vygruzka-dokumentov).

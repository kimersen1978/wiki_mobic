## **Подзапрос Service**

Назначение: Выгружает различные сервисные параметры настройки работы Моби-С. Так же выгружает список доступных отчетов и параметры их загрузки.

| Параметр | Назначение |
| --- | --- |
| ТекущаяДата | Дата загрузки данных |
| КодТипаЦены | Код цены по умолчанию (константа розничный тип цены). |
| ПериодСнятияGPSКоординат | 0- координаты на КПК сниматься не будут. |
| ПериодОтправкиGPSКоординат | не используется |
| ПечатьФормБезКода | Если 1, то все формы с КПК будут печататься с пустым полем номера документа. |
| ВключитьGPSМониторинг | 1 - включить функцию отправки GPS координат агента в реальном времени, 0 - выключить |
| **Реквизиты параметров подключаемых отчетов** |     |
| КодОтчета | Уникальный идентификатор отчета. |
| НазваниеОтчета | Название отчета для отображения в меню на КПК. |
| Выгружать | Если 1, то после полной загрузки КПК запрашивает отчет по всем клиентам участвующим в маршруте, если 0, то ничего не происходит. |
| ВыборПериода | 1 - при запросе отчета на КПК будет выбираться период. |
| ОтборПоКонтрагенту | 1 - при запросе отчета на КПК будет выведен список доступных клиентов. |
| ОтборПоДоговору | 1 - отчет формируется с учетом выбранного в шапке документа договора. Отчеты по договору можно формировать только в документах. |
| ОтборПоТовару | 1 - при запросе отчета будет выведен список доступных товаров. |
| ОтборПоСкладу | 1 - при запросе отчета будет выведен список доступных складов. |

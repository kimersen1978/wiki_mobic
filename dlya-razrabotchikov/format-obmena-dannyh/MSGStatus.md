# **MSGStatus – запрос на выгрузку статуса сообщений**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
MSGStatus\r\n
[<КодСообщения>\t<Статус>\r\n]
Ответ:
OK
\r\n
```

| Параметр | Назначение |
| --- | --- |
| КодСообщения | Код сообщения для которого выгружаем статус. |
| Статус | Текущий статус сообщения.  1-Новое 2-Отправлено, 3-Прочитано, 4-Выполнено |

Запрос MSGStatus выполняется в момент выполнения запроса Выгрузка документов.

# **FastTasks – запрос на выгрузку маршрута торгового агента**

```plaintext
<ВерсияПротоколаОбмена>\r\n
<Платформа>\r\n
<КодАгента>\r\n
FastTask\r\n
Ответ:
OK\r\n
[<КодЗадания>\t<ТипЗадания>\t<Экстра>t<ДатаВремя>\t<КодКлиента>\t<Описание>\t<Цвет>\t<ДатаОкончания>\t<Изображение>\r\n]
\r\n
```

Выгружает на КПК информацию о заданиях назначенных агенту.

| Параметр | Назначение |
| --- | --- |
| КодСобытия | Код события, уникален в пределах дня. |
| КодЗадания | Внутренний номер таблицы на КПК.<br><br>12 – Реализация<br><br>14 – Заказ<br><br>16 – Покупка<br><br>18-ПКО<br><br>19 – Анкета  <br>35 - Произвольное задание |
| Экстра | Дополнительная информация о задании. Для анкет – название формы, для Задания с фотоотчётом равно “Photo” |
| ТекущаяДата | День, в который задание будет отображаться в списке заданий. |
| КодКлиента | Код клиента в учетной системе. |
| Описание | Полное описание задания |
| Цвет | Задает цвет отображения строки задания в списке заданий на КПК. Цвет необходимо передавать в формате ARGB. [Утилита для генерации значения цвета](https://mobi-c.ru/arc/RGBtoARGB.exe).  <br>Например: Оранжевый "-23296", Розовый "-60269", Фиолетовый "-1146130",  Желтый "-256". |
| ДатаОкончания | Информационное поле. Позволяет в описании задании посмотреть конечную дату его актуальности. |
| Изображение | Список названий файлов изображений сопоставленных с заданием. Разделитель файлов US (Таблица ASCII код 31). |

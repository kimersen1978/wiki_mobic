Файл emailer.exe предназначен для отправки сообщений в отдел технической поддержки Моби-С. Для отправки данных необходимо запустить файл emailer.exe и в качестве параметра передать путь к XML файлу с параметрами и текстом для отправки. Эта утилита ни как не влияет на работу Моби-С и предназначена только для быстрой связи с разработчиками. Вы можете в любой момент отключить её использование и удалить из основного каталога Моби-С.

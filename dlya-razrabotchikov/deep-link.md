Моби-С начиная с мобильного клиента, код версии 48 имеет встроенную поддержку технологии Deep Link. Это позволяет из стороннего приложения открыть Моби-С, документ **Заказ** или **Продажа** с нужным клиентом.

Формат ссылки для запуска Продажи: `app:ru.mobis.trade/saleorder?docType=Sale&action=New&clientId=Ю00000006`

Для Заказа: `app:ru.mobis.trade/saleorder?docType=Order&action=New&clientId=Ю00000006`

Параметры запроса:

-   **docType** - тип документа. Order - заказ, Sale - продажа.
-   **action** - пока всегда New.
-   **clientId** - код клиента. Формируется в 1С:Предприятие при выгрузке справочника Контрагенты. Коды можно увидеть во время полной загрузки, если в окне обработки обмена в 1С:Предприятии поставить галочку "Включить режим отладки".

После установки apk на устройство, на котором Моби-С ранее не устанавливалась, будет создана демонстрационная база. В этой базе по указанным ссылкам будут открываться документы для клиента Алхимов. Можно поэкспериментировать, если подключить телефон к компьютеру и набрать команду

`adb shell am start -W -a android.intent.action.VIEW -d`  
`"app:ru.mobis.trade/saleorder?docType=Sale\&action=New\&clientId=Ю00000006"`

Используя HTML теги можно реализовать красивое форматирование выводимых данных.

Список реквизитов с поддержкой HTML тегов.

Запрос FullLoad, подзапрос Goods реквизит Описание.

Запрос FullLoad, подзапрос Clients реквизит Долг и Описание.

Запрос FullLoad, подзапрос Contracts реквизит Долг.

Запрос FullLoad, подзапрос SalesHelp реквизит ИсторияПродаж.

Запрос FullLoad, подзапрос DocReasons67 реквизит Комментарий.

Поддерживается большинство стандартных HTML тегов форматирования текста.

 

Пример:

В реквизит ИсторияПродаж выгружаем следующие данные

ИсторияПродаж = "
<font color=""#ff0000"">11.02=10</font>
<font color=""#FFA500""><sup> 10%</sup></font>
<br>
<font color=""#ffff00"">17.02=6</font>
<font color=""#FFA500""><sup> 12%</sup></font>
<br>
<font color=""#008000"">25.02=7</font>
<font color=""#FFA500""><sup> 18%</sup></font>
<br><font color=""#ff00ff""><b>Среднее 15% </b></font>
";

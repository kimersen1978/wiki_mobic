Запрос Обновление данных практически полностью дублирует запрос Полная загрузка, но по умолчанию он не обновляет данные по клиентам и долгам. Выполнение запроса Обновление данных не удаляет документы. Обычно агенты используют обновление данных для получения в течение дня актуальной информации по наличию товара и ценам из учётной системы.

Настройки агента вкладка Настройки КПК, флажок Выгружать клиентов при обновлении данных.  Если флажок установлен, то в запрос добавляется  информация о клиентах (Контрагенты, Договора, Папки, Документы основания). Если галка установлена, то запрос Обновление данных по сути будет равен запросу Полная загрузка, но с возможностью сохранения документов в журнале.

Программисту: С мобильного устройства отправляется запрос Rests.

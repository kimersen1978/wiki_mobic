В зависимости от выбранного клиента и договора, данный запрос позволяет загрузить из 1С персональные цены контрагента. Этот запрос похож на [Коррекцию](/mobilnyj-klient-na-android/dokumenty/zakaz-i-prodazha/korrekciya), но у него есть преимущество, он позволяет увидеть персональные цены контрагента до начала оформления документа на мобильном устройстве.

1.  Создайте документ **Продажа** или **Заявка**
2.  Нажмите аппаратную или программную кнопку **Меню** на мобильном устройстве.
3.  Выберите пункт **Запросить цены контрагента**.
4.  Начнётся обмен данными КПК с 1С.

Товары, для которых были загружены спец. цены, выделятся фиолетовым цветом. Выберите фильтр **Есть спец цены**, чтобы отобрать товары с новыми ценами.

![](/mobilnyj-klient/an-13.png)

После расчета загруженные цены контрагента сохраняются на устройстве до очередного выполнения запроса **Полная загрузка.** Цены можно использовать без повторного выполнения запроса **Запросить цены контрагента**.

Запрос обрабатывает не все типы и варианты установки персональных цен в 1С:

Для конфигураций на базе **Управление торговлей 10.3** используется документ **Установка типов цен по группам номенклатуры для покупателей,** вне зависимости от **Вида операции** документа.

Для конфигураций на базе **Управление торговлей 11** используются документы **Уточнение цен по товарам** и **Уточнение цен по ценовым группам** самого соглашения. Важно помнить, что чем точнее цена, тем приоритет выше. Поэтому при расчете сначала идет **Уточнение цен по ценовым группам**, потом **Уточнение цен по товарам**.

> Программисту: За загрузку спец цен отвечает запрос [**ClientPrices**](/dlya-razrabotchikov/format-obmena-dannyh/ClientPrices).

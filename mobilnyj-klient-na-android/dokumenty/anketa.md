Документ **Анкета** предназначен для ввода различной маркетинговой информации о клиенте. Анкета заполняется торговым агентом при посещении торговой точки клиента. С помощью анкет можно вводить любую информацию, требующую анализа в офисе. Например: ассортимент конкурентов, представленный в торговой точке, количество прилавков, холодильников, торговая площадь и др. Отключить или переименовать кнопку документа можно в [настройках агента](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nastrojki) в обработке Моби-С.

## **Вкладка Клиент**

![](/mobilnyj-klient/help71.png)

| Элемент | Назначение |
| --- | --- |
| Дата | Дата создания документа. По умолчанию подставляется текущая дата устройства. |
| Клиент | Клиент или торговая точка.  После выбора клиента отображается его фактический адрес и долг. |
| Анкета | Список доступных анкет. Список формируется в 1С в настройках агента. |
| Комментарий | Комментарий к документу. |
| Кнопка Категории | Выбрать категории документа. Присвоение определенной категории определяется наличием флага около наименования категории. Эта функция является аналогом категорий объектов в 1С 8.x. |
| Кнопка Выгрузить | Выгружает документ в 1С. |
| Кнопка Фоновая выгрузка | Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, который осуществляются каждые 15 минут. |

После выбора клиента отображается его фактический адрес и долг по выбранному договору.

## **Вкладка Анкета**

На этой вкладке отображается анкета, которую выбрали в списке анкет на вкладке **Клиент**.

![](/mobilnyj-klient/help72.png)

Анкеты создаются в визуальном редакторе форм в [серверной части Моби-С](/integraciya-s-1s-8-ut-10/merchendajzing/ankety).

## **Вкладка Товары**

Если в форме есть анкета с привязкой к номенклатуре, то в документе появляется дополнительная вкладка **Товары**. В Моби-С, начиная с версии 5.5 появилась возможность создавать анкеты с привязкой к номенклатуре. Для того, чтобы эта функция заработала, необходима доработка конфигурации 1С.

[Доработка конфигурации УТ 10.3 для анкетирования в разрезе товаров](/dlya-razrabotchikov/podsistemy/ut-10-3-anketirovanie-nomenklatury)

Нажмите на товар, чтобы открыть анкету.

![](/mobilnyj-klient/goodsanket2.png)

Назначение кнопок анкеты

**Отмена** - выход из формы без сохранения.

**Очистить** - очистить все ранее введенные значения формы.

**ОК** - сохранить введенные значения и закрыть форму.

![](/mobilnyj-klient/goodsanket3.png)

## **Меню документа**

Для вызова меню документа нажмите кнопку меню (3 точки) или аппаратную кнопку **Меню**. Содержимое меню зависит от выбранной вкладки.

| Элемент | Назначение |
| --- | --- |
| Выгрузить | Отправить документ на фоновую выгрузку. Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, которые осуществляются каждые 15 минут. |
| Запросить отчет | Загрузка или просмотр [отчетов](https://mobi-c.ru/help/mobi_androidreports.htm). |
| Печать | [Распечатать](https://mobi-c.ru/help/mi8_serviceprintform.htm) или отправить печатную форму по email. |
| Штрихкод | Только для вкладки Товары. Открывает встроенный сканер штрихкода. Позволяет искать товар по [штрихкоду](https://mobi-c.ru/help/mobi_barcode.htm). |
| Запросить историю продаж | Только для вкладки Товары. При выполнении этого запроса из учётной системы запрашиваются данные по [истории продаж](https://mobi-c.ru/help/mobi_androidrequesthistory.htm) для выбранного клиента. |

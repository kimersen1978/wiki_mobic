ПКО - документ для приема денег от клиентов. Отключить или переименовать кнопку документа можно в [настройках](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nastrojki) агента в обработке Моби-С.

![](/mobilnyj-klient/help52.png)

|     |     |
| --- | --- |
| Дата | Дата создания документа. По умолчанию подставляется текущая дата устройства. |
| Флажок Пр | Не используется. Можно задействовать под собственные нужды. Например можно передать признак проводить документ или нет при выгрузке. |
| Фирма | Фирма, от которой мы работаем. При выборе клиента или договора фирма может автоматически измениться. Это происходит если у клиента задана фирма по умолчанию. |
| Клиент | Клиент или торговая точка. После выбора клиента отображается его фактический адрес и долг по выбранному договору. |
| Договор | Выбор договора взаиморасчетов с клиентом. После выбора клиента по умолчанию встает договор с названием Авто. Договор Авто будет обработан при выгрузке документа в 1С, обычно вместо него в документ подставляется договор по умолчанию для данного клиента. При выборе договора может измениться фирма, тип цены и скидка. |
| Основание | Позволяет выбрать документ-основание (или несколько документов) для платежа. Если выбрано несколько документов, то суммы оснований суммируются и общая сумма подставляется в поле Сумма. |
| Сумма | Итоговая сумма ПКО. |
| Комментарий | Комментарий к документу. |
| Кнопка Выгрузить | Выгружает документ в 1С. |
| Кнопка Фоновая выгрузка | Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, который осуществляются каждые 15 минут. |

## **Выбор документов-оснований**

Документы основания выгружаются из 1С. Так же в качестве оснований можно использовать созданные на кпк документы **Заказ** и **Продажа**. По умолчанию отображаются все доступные документы основания. Выбрать можно только документы основания, для установленного в поле **Договор** договора.

![](/mobilnyj-klient/help52.png)

## **Меню документа**

Для вызова меню документа нажмите кнопку меню (3 точки) или аппаратную кнопку **Меню**. Содержимое меню зависит от выбранной вкладки.

|     |     |
| --- | --- |
| Печать | Распечатать или отправить печатную форму по email. Если подключен ККМ появляется дополнительная печатная форма **Чек ККМ (ПКО)**. |
| Выгрузить | Отправить документ на фоновую выгрузку. Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, которые осуществляются каждые 15 минут. |
| Запросить отчет | Загрузка или просмотр отчетов. |

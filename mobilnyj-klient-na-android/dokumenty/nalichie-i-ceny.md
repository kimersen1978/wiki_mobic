Документ **Наличие и цены** имеет несколько предназначений. Основное это проведение инвентаризации на складе или в торговой точке. Так же с его помощью можно проводить маркетинговые исследования и собирать информацию о ценах на товары. Отключить или переименовать кнопку документа можно в [настройках](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nastrojki) агента в обработке Моби-С.

## **Вкладка Параметры**

![](/mobilnyj-klient/help73.png)

| Элемент | Назначение |
| --- | --- |
| Дата | Дата создания документа. По умолчанию подставляется текущая дата устройства. |
| Склад | Выбор склада, на котором проводится учет. |
| Клиент | Клиент или торговая точка. |
| Тип цен | Выбор типа цены. |
| Комментарий | Комментарий к документу. |
| Фото | Возможность создать фото к документу |
| Кнопка Категории | Выбрать категории документа. Присвоение определенной категории определяется наличием флага около наименования категории. Эта функция является аналогом категорий объектов в 1С 8.x. |
| Кнопка Выгрузить | Выгружает документ в 1С. |
| Кнопка Фоновая выгрузка | Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, который осуществляются каждые 15 минут. |

## **Вкладка Товары**

Табличную часть документов можно настроить, изменив отображаемые данные.

![](/mobilnyj-klient/help75.png)

Содержимое и функциональность аналогично документу [**Заказ**](/mobilnyj-klient-na-android/dokumenty/zakaz-i-prodazha/nalichie-i-ceny).

## **Диалог ввода количества**

При нажатии на строку списка товаров открывается диалог ввода количества.

![](/mobilnyj-klient/help74.png)

| Элемент | Назначение |
| --- | --- |
| Единица | Выбор единицы товара. Список единиц и единица **по умолчанию** задаются в [настройках агента](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nomenklatura) в обработке Моби-С. |
| Тип цены | При выборе значений из этого списка будет изменяться реквизит **Цена**. Тип цены предназначен только для изменения цены и не выгружается в 1С. По умолчанию стоит тип цены выбранный на вкладке **Клиент**. Если список активен значит торговому агенту разрешено изменять цену товара. |
| Цена | Текущая цена товара за выбранную единицу. |
| Кол-во | Поле предназначено для ввода количества реализуемого товара. При смене единицы введённое количество будет пересчитано в новую единицу. **Дробное количество** можно вводить для товаров у которых установлен признак [**Весовой**](/voprosy-i-otvety/vvod-drobnogo-kolichestva) в 1С. Для всех остальных дробная часть при сохранении будет сбрасываться. |
| [Дополнительные реквизиты](/dlya-razrabotchikov/dopolnitelnye-rekvizity) | Для каждого документа есть возможность создать персональный набор дополнительных реквизитов, которые будут отображаться в карточке товара. |

## **Поиск товара и меню документа**

Поиск товара и работа с меню документа производиться аналогично работе с документом [Заказ](https://mobi-c.ru/help/mobi_androidsearch.htm).

## **Меню документа**

Для вызова меню документа нажмите кнопку меню (3 точки) или аппаратную кнопку **Меню**. Содержимое меню зависит от выбранной вкладки.

| Пункты меню | Назначение |
| --- | --- |
| Выгрузить | Отправить документ на фоновую выгрузку. Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, которые осуществляются каждые 15 минут. |
| Запросить отчет | Загрузка или просмотр отчетов. |
| Печать | Распечатать или отправить печатную форму по email. |
| Штрихкод | Открывает встроенный сканер штрихкода. Позволяет искать товар по штрихкоду. |
| [Запросить коррекцию документа](/mobilnyj-klient-na-android/dokumenty/zakaz-i-prodazha/korrekciya) | Коррекция отправляет текущий документ в 1С, где он обрабатывается с использованием всех алгоритмов ценообразования 1С. Торговый агент увидит различия между документом на КПК и тем, что получится после выгрузки. Торговый агент может принять эти различия, либо оставить как есть. |
| Вид | Выбор вида отображения табличной части на вкладке Товары: список, список с иконками или в виде эскизов. |

Документ **Перемещение** предназначен для оформления перемещения товара со склада на склад. Если в настройках агента включена настройка [Режим торговля с колес](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nastrojki-kpk), то добавляются два документа **Загрузка** и **Разгрузка.** Они полностью идентичны документу **Перемещение,** но у них заданы по умолчанию склады загрузки и разгрузки. Отключить или переименовать кнопку документа можно в [настройках агента](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nastrojki) в обработке Моби-С.

## Вкладка **Параметры**

![](/mobilnyj-klient/help76.png)

| Элемент | Назначение |
| --- | --- |
| Дата | Дата создания документа. По умолчанию подставляется текущая дата устройства. |
| Флажок Пр. | Функционал флага зависит от конфигурации 1С. Обычно используется для отражения документа в бух. и нал. учетах. |
| Откуда - Фирма | Фирма, от которой производиться отгрузка товара. |
| Откуда - Склад | Склад, с которого производится отгрузка товара. |
| Куда - Фирма | Фирма, на которую поступает товар. |
| Куда - Склад | Склад, на который поступает товар. |
| Кнопка Выгрузить | Выгружает документ в 1С. |
| Кнопка Фоновая выгрузка | Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, который осуществляются каждые 15 минут. |

## **Вкладка Товары**

![](/mobilnyj-klient/help77.png)

Табличную часть документов можно настроить, изменив отображаемые данные.

## **Диалог ввода количества**

При нажатии на строку списка товаров открывается диалог ввода количества.

![](/mobilnyj-klient/help78.png)

|     |     |
| --- | --- |
| Элемент | Назначение |
| Единица | Выбор единицы товара. Список единиц и единица **по умолчанию** задаются в настройках агента в обработке Моби-С. |
| Тип цены | При выборе значений из этого списка будет изменяться реквизит **Цена**. Тип цены предназначен только для изменения цены и не выгружается в 1С. По умолчанию стоит тип цены выбранный на вкладке **Клиент**. Если список активен значит торговому агенту разрешено изменять цену товара. |
| Цена | Текущая цена товара за выбранную единицу. Возможность редактирования цены задается в настройках агента. |
| Наличие | Остаток товара по выбранному складу, в выбранной единице. Вид отображения остатка можно изменить в настройках агента. |
| Кол-во | Поле предназначено для ввода количества реализуемого товара. При смене единицы введённое количество будет пересчитано в новую единицу. **Дробное количество** можно вводить для товаров у которых установлен признак Весовой в 1С. Для всех остальных дробная часть при сохранении будет сбрасываться. |
| Дополнительные реквизиты | Для каждого документа есть возможность создать персональный набор дополнительных реквизитов, которые будут отображаться в карточке товара. |

Под полями ввода отображается произвольная информация о товаре. Эта информация задается в реквизите **Описание** подзапроса [Goods](https://mobi-c.ru/help/mobifullload.htm#goods). Ниже расположены фото товара и кнопка для добавления новых фото с помощью встроенной камеры мобильного устройства.

## **Поиск товара и меню документа**

Поиск товара и работа с меню документа производиться аналогично работе с документом Заказ.

## **Меню документа**

Для вызова меню документа нажмите кнопку меню (3 точки) или аппаратную кнопку **Меню**. Содержимое меню зависит от выбранной вкладки.

|     |     |
| --- | --- |
| Пункты меню | Назначение |
| Выгрузить | Отправить документ на фоновую выгрузку. Сохраняет и закрывает документ. Документ выгрузится в 1С при следующем фоновом сеансе связи, которые осуществляются каждые 15 минут. |
| Запросить отчет | Загрузка или просмотр отчетов. |
| Печать | Распечатать или отправить печатную форму по email. |
| Штрихкод | Открывает встроенный сканер штрихкода. Позволяет искать товар по штрихкоду. |
| Запросить коррекцию документа | Коррекция отправляет текущий документ в 1С, где он обрабатывается с использованием всех алгоритмов ценообразования 1С. Торговый агент увидит различия между документом на КПК и тем, что получится после выгрузки. Торговый агент может принять эти различия, либо оставить как есть. |
| Вид | Выбор вида отображения табличной части на вкладке Товары: список, список с иконками или в виде эскизов. |

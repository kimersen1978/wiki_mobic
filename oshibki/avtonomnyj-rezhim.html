Для того чтобы мобильные агенты могли загружать данные из 1С и выгружать созданные документы необходимо, чтобы в офисе был запущен модуль интеграции Моби-С и в нем разрешен обмен с КПК. На главном экране на вкладке Сервер обмена смотрите параметр Режим обмена с КПК.

Как включить в модуле интеграции обмен с КПК?

При открытии модуля интеграции Моби-С пишет Режим обмена с КПК Выключен. С права от надписи через дефис указана причина по которой выключен режим обмена с КПК.

 

Пользователь 1С не авторизован на обмен с КПК

В настройках модуля интеграции Моби-С указывается список пользователей 1С для которых разрешен обмен данными с КПК. Если обработка запущена под пользователем 1С которого нет в списке, то режим обмена с КПК выключиться. Вы можете добавить текущего пользователя в список авторизованных пользователей.

 

Ошибка подключения библиотеки PocketServer.dll

При запуске модуля интеграции Моби-С происходит подключение к внешней компоненте PocketServer.dll.

На некоторых версию Windows потребуется ручная регистрации PocketServer.dll.

 

Отсутствует доступ к Основному каталогу Моби-С

Недоступен ранее заданный Основной каталог Моби-С. Рекомендуем использовать сетевой путь к Основному каталогу Моби-С.

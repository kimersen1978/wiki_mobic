При выполнении запроса Полная загрузка или Загрузка фотографий в логе обмена красные строки Ошибка Файл не найден.

Ошибка указывает на проблемы с файлами фотографий. При выполнении запроса Полная загрузка формируется список фотографий, которые необходимо выгрузить на КПК. В конце запроса в папке агента \Photo\Device\New из базы 1С выгружаются файлы фотографий по заранее сформированному списку. Однако, по некоторым причинам, количество выгруженных файлов может быть меньше, чем указано в списке. Ошибка Файл не найден сообщает о количестве отсутствующих файлов.

Названия ошибочных файлов сохраняются в лог-файле на КПК, который затем выгружается в 1С. По этим названиям с помощью специальной обработки можно найти объект (товар или контрагента), связанный с файлом.

Как найти проблемные файлы?

После того как на КПК была получена ошибка Файл не найден:

1. На КПК выполните запрос Обмен данными ⇒ Выгрузить документы, чтобы выгрузить лог-файл. В настройках убедитесь, что выбран Уровень логирования на КПК: "Полный лог", "Ошибки" или "Ошибки, Действия агента, Системные события". В противном случае в логе не будет нужных данных.

2. В обработке Моби-С перейдите на вкладку Агенты  и нажмите кнопку Лог действий агента. Выберите нужного агента, установите период и отметьте галочку Ошибки, затем нажмите кнопку Сформировать.

Будут выведены имена всех проблемных файлов. Далее необходимо по названию найти объект и само фото в 1С. Для этого потребуется скачать специальную сервисную обработку Моби-С.

3. Скачайте обработку ПолучитьВладельцаФайла (УФ).epf (конфигурации УТ11, КА2, ERP, УНФ, БП3, Розница3) или ПолучитьВладельцаФайла (ОФ).epf (конфигурации УТ10.3, УПП, УТП, КА1, БП2).

4. Запустите обработку в вашей 1С через Файл ⇒ Открыть.

В поле Имя файла введите скопированное из лога имя файла и нажмите кнопку Найти владельца.

После поиска в поле Владелец фото будет отображён объект, в котором находится проблемное фото, а в поле Фото — имя и ссылка на фото.

Наиболее распространённой причиной проблем с фото является битая ссылка на файл или файл некорректного формата. Проблемный файл можно удалить или заменить на корректный.

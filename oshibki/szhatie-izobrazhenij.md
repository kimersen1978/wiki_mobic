## **На данном компьютере невозможно выполнить сжатие изображений для выгрузки на КПК.**

Проблема со сжатием может появляться на старых системах Windows XP или 2003 сервере.

Для работы с изображениями мы используем WIA: для чтения файла используется объект WIA.ImageFile для преобразования используется WIA.ImageProcess

В Вашем случае в системе не установлена необходимая компонента.

Инструкция по установке компоненты:

**1.**переименовать файл "[wiaaut.dll\_](https://mobi-c.ru/arc/wiaaut.dll_)" в **wiaaut.dll**

**2.**скопировать файл **wiaaut.dll** в **<Каталог Windows>\\System32 или SysWOW64**

**3.**запустить консоль (**cmd**) с правами **Администратора**, перейти в каталог куда скопировали **wiaaut.dll**

**4.**выполнить регистрацию командой **regsvr32 wiaaut.dll**

Если вы получаете данное сообщение на сервера более позднего выхода. Для включения сжатия достаточно включить компоненту **Возможности рабочего стола**.

![](/error/sjatie_serv_2008.png)![](/error/sjatie_2012.png)

После выполнения указанной инструкции в обработке Моби-С  [**Сервис ⇒ Параметры ⇒ Прочее**](/integraciya-s-1s-8-ut-10/servis/parametry/prochee) должна стать активной ссылка **Настройка выгрузки изображений на КПК.**

Устанавливайте параметры сжатия, очищайте каталог фотографий тестового агента (**Настройки агента ⇒ Настройки ⇒** [**Фотографии**](/integraciya-s-1s-8-ut-10/agenty/nastrojki-agenta/nastrojki) **⇒ Очистить фотографии на КПК**) и выполняйте с КПК полную загрузку и смотрите на загрузку фотографий.

Остатки и резерв товара

Для того что бы посмотреть текущие остатки по товару в вашей 1С, открываем в карточке товара отчет Свободные остатки.

Главное ⇒ Номенклатура ⇒ Карточка товара ⇒ Отчеты ⇒ Свободные остатки

  • Столбец Свободно - остаток товара на складе с / без учета резерва
  • Столбец Резерв - резерв товара на складе.
  • Столбец Остаток - остаток товара на складе с учетом резерва.

После того как сформировали отчет, выполните на КПК запрос Полная загрузка и сравните остаток в КПК и 1С.

Параметры влияющие на остаток на КПК

1. Настройки агента ⇒ вкладка Номенклатура ⇒ настройка Учитывать резерв при расчете остатков

  • Если флажок установлен, то смотреть столбец отчета Свободно.
  • Если флажок снят, то смотреть столбец отчета Остаток.

2. Настройки агента ⇒ вкладка Склады ⇒ список доступных складов. На КПК склад Авто показывает сумму остатка по всем доступных складам.

3. Настройки агента ⇒ вкладка Склады ⇒ настройка Отображать остаток в разрезе организаций. Поддержка остатков по складам в разрезе организаций реализована в 86 версии мобильного клиента и обработке 5.5.38.

Долг клиента

Общий долг клиента

Для того что бы посмотреть общий долг клиента в вашей 1С, открываем в карточке клиента отчет Взаиморасчеты с контрагентом.

Главное ⇒ Покупатели ⇒ Карточка покупателя ⇒ Отчеты ⇒ Взаиморасчеты с контрагентом.

Столбец Конечный остаток - задолженность клиента.

Просроченный долг клиента

Для того что бы посмотреть просроченный долг клиента в вашей 1С, открываем в карточке клиента отчет Задолженность покупателя по срокам долга.

Главное ⇒ Покупатели ⇒ Карточка покупателя ⇒ Отчеты ⇒ Задолженность покупателя по срокам долга.

Столбец В т.ч. просроченная задолженность.

Параметры влияющие на долг на КПК

Настройки агента ⇒ вкладка Организации ⇒ Список организаций. В КПК долг клиента выгружается общий для организаций доступных агенту.

Внимание! Долги в отчете отображаются в валюте документа. На КПК долг пересчитывается в валюту заданную в Настройки агента ⇒ вкладка Настройки > настройка Валюта на КПК. Курс пересчета находится в Регистр сведений > Курсы валют.

Документы основания для ПКО и архива документов

Выгружаются документы если у контрагента в настройках взаиморасчетов стоит галка Долги по документам.

Главное ⇒ Покупатели ⇒ Карточка покупателя ⇒ Отчеты ⇒ Взаиморасчеты с контрагентом.

Параметры влияющие на выгрузку документов оснований

1. Выгружаются только документы Заказ и Реализация.

2. Настройки агента ⇒ вкладка Настройки ⇒ кнопка ПКО, отроется диалог Настройки документа ПКО

  • Список выбора Создавать, значение "Приходный кассовый ордер", если выбрано значение “Запрещен для ввода с КПК”, то документы основания не выгружаются.
  • Список выбора Период -  задает период выборки документов.
  • Флажок Только документы агента  - только документы в которых на вкладке Дополнительно в поле Ответственный выбран текущий торговый агент.

3. Настройки агента ⇒ вкладка Склады - только документы на склады из списка.

4. Настройки агента ⇒ вкладка Организации - только документы на организации из списка.

5. Настройки агента ⇒ вкладка Типы цен - только документы на типы цен из списка.

6. Договор из документа должно быть доступно агенту.

Если по всем этим параметрам документ должен быть доступен необходимо проверить корректность заполнения клиента, организации, договора и не помечен ли на удаление договор.

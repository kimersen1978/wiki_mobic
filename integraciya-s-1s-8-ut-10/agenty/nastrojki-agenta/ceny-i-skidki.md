На вкладке **"Цены и скидки"** типы цен и скидок, которые будут выгружаться на КПК.

![](/ut10/ut10-3.png)

## **Блок Типы цен**

В данном блоке настраиваются типы цен, которые будут выгружаться на КПК.

Поле **Тип цен по умолчанию** (обязательно к заполнению) - Значение типа цены **по умолчанию**. Цена по умолчанию используется на КПК тогда, когда информация о типе цен недоступна, например, когда не выбран клиент.

Флажок **Типы цен из договоров** \- Если признак установлен, то Моби-С выгружает на КПК только **типы цен,** которые встречаются в выгружаемых договорах контрагентов.

Ниже в таблице перечислен список типов цен, зарегистрированных в системе. Если флажок **Типы цен из договоров** снят, то на КПК будут выгружены только отмеченные галкой **типы цен** и **договора** с этими или не установленным типом цен.

## **Блок Типы скидок**

В данном блоке настраиваются скидки, которые будут выгружаться на КПК. Скидка это только **процент** скидки или наценки, который можно использовать в шапке или табличной части документа на КПК.

Флажок **Выгружать установленные в 1С скидки** - Если признак установлен, тогда Моби-С выберет все проценты скидок, которые встречаются в документе **Установка скидок номенклатуры** и выгрузит их на КПК. Используются только документы без каких либо условий. Настройки, заданные в списке скидок, игнорируются.

Список скидок содержит данные из справочника **Типы скидок и наценок**. Для того, чтобы выгрузить на КПК конкретные скидки, снимите флажок **Выгружать установленные в 1С скидки** и выберите в списке нужные скидки.

Флажок **Выгружать уточнения цен по группам номенклатуры**, для каждого контрагента генерируется цена с учетом документа **Установка типов цен по группам номенклатуры для покупателей**. Новый тип цен устанавливается для клиента "по умолчанию".

## **Расчет наценки**

Функция [**Расчет наценки**](/funkcii/raschet-nacenki) позволяет выгрузить на КПК для всех товаров цену, которая считается закупочной. Зная закупочную и продажную цену, можно оценить доходность продажи товара или всего документа.

●**Не доступно на КПК** - функция **Закупочная цена** не используется.

●**Равна типу цены** - выбрать конкретный тип цен из справочника **Типы цен номенклатуры**.

●**\[Тип цены\]** - последнее выбранное значение из справочника **Типы цен номенклатуры**.

Флажок **"Запретить изменят цены и скидки на КПК"**, установлен, то запрещено редактирование цен и скидок на КПК для всех клиентов, либо только для выбранных категорий. Нажмите кнопку с тремя точками что-бы открыть диалог выбора категорий. Выберите необходимые категории, присвойте категории клиентам, для которых нужно запретить изменение.

Если флажок сброшен, то разрешено редактирование для всех клиентов. Настройка действует для документов "**Заказ**", "**Продажа**", "**Покупка**".

Флажки **"Разрешать изменять цену, тип цены, скидку"**, разрешают редактировать цену, выбирать из разрешенных типов цен и скидок данные в карточке товара на КПК для всех товаров, либо только для выбранных категорий. Нажмите кнопку с тремя точками что-бы открыть диалог выбора категорий. Выберите необходимые категории, присвойте категории товарам, для которых нужно разрешить изменение.

По умолчанию все флажки сброшены и редактирование цены и скидки в карточке товара запрещены. Настройка действует для документов "**Заказ**" и "**Продажа**".=

> Внимание! Настройка "Запретить изменят цены и скидки на КПК" имеет приоритет.

Журнал фотографий содержит:

  • Фотографии товаров выгружаемые на мобильные устройства.
  • Фотографии клиентов выгружаемые на мобильные устройства.
  • Фотографии выгружаемые вместе с заданиями.
  • Фотографии товаров загруженные с мобильных устройств.
  • Фотографии клиентов загруженные с мобильных устройств.
  • Фотографии созданные в документах.
  • Фотографии созданные в заданиях.

Описание столбцов списка фотографий

Двойной клик на эскизе фотографии открывает встроенный просмотрщик фотографий.

Кнопка со значком "информация" открывает дополнительные данные по фотографии.

Фото корректно - показывает корректность фото по GPS, если данных по GPS нет, то вы увидите красный крестик и сообщение вида "GPS-время съёмки неизвестно."

Цифровая подпись - показывает корректность цифровой подписи фотографии. Если фотография подвергалась изменением на кпк, то цифровая подпись будет некорректна.

Дата съемки - дата и время создания фотографии.

Показать на карте - отображение место создания фотографии на карте Гугл.

Комментарий - отображает комментарий, если он был добавлен перед сохранением фотографии.

Перейти к файлу - открывает папку в проводнике Windows куда была выгружена фотография.

Добавить в 1С - Добавляет выбранное фото в карточку товара или клиента в соответствующем справочнике.

Настройки отчета

Настройки отчета позволяют задать отбор для списка фотографий.

Общие настройки

В шапке формы вы можете задать период, за который необходимо отобразить фотографии.

Поле выбора Отображать фотографии - задает один из вариантов отображения (все, только корректные, только некорректные).

Флажок Отображать счетчики фотографий - добавляет в отчет 3 колонки (ош - фотографии с ошибками, некорректные, вх - входящие, ис - исходящие).

Поле Количество эскизов в строке отчета - задает максимальное количество эскизов которые будут отображаться в поле Эскизы списка фотографий. Если установить значение 0, то поле Эскизы будет скрыто.

Поле Максимальное количество фотографий - задает количество фотографий выводимых в отчете.

Отображать фотографии

Позволяет фильтровать отображение фотографий в отчете по привязке к тем или иным объектам 1С.

Отбор

Позволяет устанавливать дополнительные отборы по агентам, документам, контрагентам и номенклатуре.

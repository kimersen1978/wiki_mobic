Вкладка Мерчендайзинг - кнопка Плановая матрица

Плановая матрица предназначена для создания ассортиментной матрицы товара. Плановая матрица это расширенный вариант функции Форматы торговых точек. Основное отличие, это фиксация изменений матрицы во времени. С помощью данной функции вы сможете назначить для клиента различную ассортиментную матрицу товаров на каждый день и у вас будет история изменения этой матрицы. Плановая матрица имеет приоритет над матрицей заданной через форматы торговых точек. Если заданы оба типа матриц, то на мобильном устройстве будет отображаться только товар из плановой матрицы.

Диалоговое окно для работы с плановыми матрицами

  1. Выбираем период планирования или просмотра данных по плановым матрицам.
  2. Выбираем агента
  3. Нажимаем кнопку Заполнить.

Поле Формат ТТ позволяет отфильтровать контрагентов по заданному формату.

Двойной клик на ячейке контрагент/дата  открывает диалоговое окно формирования плановой матрицы на выбранную дату.

Поле Контрагент -  отображается контрагент для которого формируется ассортиментная матрица.

Поле Формат ТТ - позволяет назначить или изменить формат торговой точки контрагента.

Список товаров входящих в ассортиментную матрицу. Нажмите на кнопку Подбор, что бы в ручную выбрать товары. Нажмите на кнопку Заполнить, что бы добавить или удалить товары из форматов торговых точек.

Параметры

Плановая матрица - SKU (товарных позиций) в формате торговой точки клиента и в плановой матрице.

MML Матрица - можно задать MML (Minimum Must List) для плановой матрицы. Сейчас этот параметр не используется.

Минимальная сумма - позволяет задать минимальную сумму для документов Заказ и Продажа. Сейчас этот параметр не используется. Минимальная сумма задается в настройках агента и действует для всех клиентов.

Содержимое ячеек можно копировать. По клику правой клавиши откроется контекстное меню.

Копировать - скопировать матрицу в буфер обмена.

Вставить  - вставить ранее скопированную матрицу из буфера обмена.

Удалить - удалить матрицу за текущий день.

Столбец Формат - отображает текущий формат торговой точки контрагента.

Столбец История - отображает параметры последней плановой матрицы. Основное назначение этого столбца это предоставление возможности посмотреть и скопировать содержимое последней плановой матрицы.

Данные плановой матрицы хранятся в документе План продаж.

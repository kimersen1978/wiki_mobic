Остатки и резерв товара

Для того что бы посмотреть текущие остатки и резерв по товару в вашей 1С, открываем отчет Анализ доступности товаров не складах.

Отчеты ⇒ Запасы (склад) ⇒ Анализ доступности товаров не складах

  • Столбец Свободный остаток - остаток товара на складе
  • Столбец Зарезервировано - резерв товара на складе
  • Столбец Остаток - общий остаток с учетом резерва.

После того как сформировали отчет, выполните на КПК запрос Полная загрузка и сравните остаток в КПК и 1С.

Если для товара включена детализация по характеристикам или используется настройка, Настройки агента > вкладка Номенклатура > настройка Детализировать на КПК по сериям необходимо в отчете добавить группировку по Сериями и Характеристикам.

Параметры влияющие на остаток на КПК

1. Настройки агента ⇒ вкладка Склады ⇒ настройка Учитывать резерв при расчете остатков

  • Если флажок установлен, то смотреть столбец отчета Свободный остаток.
  • Если флажок снят, то смотреть столбец отчета  Остаток.

2. Настройки агента ⇒ вкладка Склады ⇒ список доступных складов. На КПК склад Авто показывает сумму остатка по всем доступных складам.

Остатки товара в разрезе организаций

Если в настройках агента включена настройка Отображать остаток в разрезе организаций, то используется отчет Ведомость по товарам и ГТД организаций.

Отчеты ⇒ Запасы (склад) ⇒ Ведомость по товарам и ГТД организаций

  • Столбец Кон. Остаток - остаток товара на складе, резервы в этом режиме не учитываются.

Поддержка остатков по складам в разрезе организаций реализована в 86 версии мобильного клиента и обработке 5.5.42.

Долг клиента

Для того что бы посмотреть общий долг клиента в вашей 1С, открываем в карточке клиента отчет Ведомость по взаиморасчетам.

Справочник контрагенты ⇒ Карточка контрагента ⇒ Отчеты ⇒ Ведомость по взаиморасчетам

Внимание! В КПК долг клиента выгружается общий, без учета разделения по фирмам.

Внимание! Долги для отчета отображаются в валюте документа. На КПК долг пересчитывается в валюту заданную в Настройки агента > вкладка Настройки > настройка Валюта на КПК. Курс пересчета находится в Регистр сведений > Курсы валют.

Просроченный долг

Типового отчета нет, но можно использовать Отчет по кредитной линии, неоплаченные документы и сроки оплаты

Отчеты ⇒ Продажи ⇒ Взаиморасчеты ⇒ Отчет по кредитной линии

Документы основания для ПКО и архива документов

Долг по документу выгружается на момент выполнения полной загрузки/обновления данных, если по документу была частичная оплата, сумма долга по документу будет с учетом частичной оплаты.

Справочник контрагенты ⇒ Карточка контрагента ⇒ Отчеты ⇒ Ведомость по взаиморасчетам

 

Параметры влияющие на выгрузку документов оснований

1. Выгружаются документы если в договоре стоит галка Вести по документам расчетов с контрагентами.

2. Выгружаются только документы Заказ и Реализация.

3. Настройки агента ⇒ вкладка Настройки ⇒ кнопка ПКО, отроется диалог Настройки документа ПКО

  • Список выбора Создавать, значение "Приходный кассовый ордер", если выбрано значение “Не доступно на КПК”, то документы основания не выгружаются.
  • Список выбора Период -  задает период выборки документов.
  • Флажок Только документы агента  - только документы в которых на вкладке Дополнительно в поле Ответственный выбран текущий торговый агент.

4. Настройки агента ⇒ вкладка Организации - только документы на организации из списка.

Если по всем этим параметрам документ должен быть доступен необходимо проверить корректность заполнения клиента, организации, договора/соглашения и не помечен ли на удаление договор/соглашение.

Внимание! Долги для отчета отображаются в валюте документа. На КПК долг пересчитывается в валюту заданную в Настройки агента > вкладка Настройки > настройка Валюта на КПК. Курс пересчета находится в Регистр сведений > Курсы валют.

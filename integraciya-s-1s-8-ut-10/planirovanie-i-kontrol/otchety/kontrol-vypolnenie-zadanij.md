Отчет Контроль выполнения заданий предназначен для контроля выполнения запланированных торговому агенту заданий и посещений клиентов, а так же он позволяет учитывать внеплановые визиты и созданные документы. Посещение клиента фиксируется на основе выполнения заданий.

Нажмите кнопку **Настройка** для задания различных параметров фильтрации и отображения отчета. После настройки отчета нажмите в верхнем тулбаре кнопку Сформировать.

![](/ut10/40.png)

## **Описание столбцов отчета**

Содержание первого столбца зависит от установленных настроек. Этот столбец устанавливает детализацию по агенту, дате, контрагенту группе, заданиям и пр. Задается в настройках отчета на вкладке Все настройки. Если задание выполнено, то добавляется дополнительная строка с результатом, документ или фотография.

### **Группа столбцов Выполнение заданий**

Столбец **Вып.** - Если значение не задано значит задание не выполнено. Выполненное задание обозначается значком V.

Столбец **План** \- Количество запланированных заданий.

Столбец **Факт** \- Количество выполненных запланированных заданий.

Столбец **Вне плана** - Количество выполненных торговым агентом заданий которые не были для него запланированы. Внеплановыми считаются документы созданные "напрямую", без задания.

Столбец **Не вып.** - Количество не выполненных запланированных заданий.

Столбец **%** - процент выполнения запланированных заданий.

Столбец **Отчет** \- Содержимое поля комментарий. Актуально только для произвольных заданий и заданий с фотоотчетом.

### **Группа столбцов Посещение**

Предоставляет данные для анализа выполнения запланированных и внеплановых посещений клиентов.

Столбец **План** \- Количество запланированных посещений клиентов.

Столбец **Факт** \- Количество посещенных, запланированных клиентов.

Столбец **Вне плана** - Количество посещенных клиентов, вне плана.

Столбец **Не посещен** - Количество не посещенных, запланированных клиентов.

Столбец **%** - процент посещенных, запланированных клиентов

Столбец **Время посещения** - временем посещения клиента является время создания документа. Время создания фиксируется при первом открытии документа.

Столбец **Время на оформление** - время которое торговый агент потратил на создание документа. Считается от открытия документа до первого сохранения. Значение 0 говорит о том что документ оформлялся менее минуты.

## **Контекстное меню**

Двойной клик мыши на ячейке отчета открывает контекстное меню.

![](/ut10/41.png)

С помощью контекстного меню можно, открыть документ, изменить сортировку и группировку отчета и пр.

## **Настройка отчета**

Нажмите в верхнем тулбаре кнопку **Настройки**.

На вкладке **Параметры**, задаем период выборки данных для отчета.

![](/ut10/25.png)

На вкладке **Отбор**, возможно задание параметров фильтрации данных отчета. Отбор данных возможен по любому столбцу отчета.

![](/ut10/26.png)

На вкладке **Все настройки** можно изменить практически любые параметры отчета. Подробно использование этой вкладки описано в справочной системе 1С.

Для того что бы автоматически сохранялись все настройки отчета, необходимо в верхнем тулбаре формы отчета нажать кнопку Действия, выбрать пункт меню Сохранить значения. Откроется диалог **Сохранение настроек**. В нем необходимо установить флажок Сохранять автоматически и нажать кнопку **ОК**.

![](/ut10/33.png)

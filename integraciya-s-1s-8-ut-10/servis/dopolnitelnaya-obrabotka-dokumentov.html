Cпециалист знакомый с программированием в 1С, может реализовать дополнительную обработку выгруженных с КПК документов без доработки обработки Моби-С. Обработки добавленные в этом модуле хранятся в базе 1С и не зависят от текущего модуля Моби-С.

Нажмите кнопку Добавить что бы добавить новую обработку.

Параметры обработки

  1. Поле Описание - название дополнительной обработки.
  2. В спиcке С КПК выгружен выберите требуемый тип документа Моби-С.
  3. В списке В 1С применить к выберите требуемый тип документа в 1С.
  4. Поле Доступно агентам, позволяет задать список агентов для которых будет выполняться обработка. По умолчанию обработка выполняется для всех агентов.
  5. Флажок Разрешить в Моби-С.Заказы - дополнительная обработка используется для документов загруженных из Моби-С и Моби-С.Заказы.
  6. Ниже в поле ввода введите код постобработки и нажмите кнопку Сохранить.

Дополнительная обработка работает только если установлен флажок Исп.

Скачать примеры дополнительных обработок

  1. Нажмите кнопку Добавить из файла
  2. Выберите загруженный файл с примерами

При обработке выгруженного с КПК значения доступны следующие переменные:

Агент - агент выгружающий документ (версия обработки 5.5.30 и выше)

Док - ДокументОбъект - Если дополнительный обработка привязана к документу или к табличной части документа

СтрДок - Строка табличной части Товары документа к которому привязана дополнительная обработка

НовыйКлиент - СправочникОбъект.Контрагенты если дополнительная обработка привязана к новому клиенту.

Обработка позволяет изменить реквизиты документа или заполнить дополнительные значения при выгрузке документа с КПК, выполняется после заполнения всех типовых и обработки дополнительных реквизитов перед сохранением/проведением документа.

Изменить типовые параметры проведения документа в 1С невозможно.

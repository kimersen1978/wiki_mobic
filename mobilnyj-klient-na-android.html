Главный экран программы Моби-С

В верхней части находится заголовок и отображается текущая версия программы. При нажатии на три точки справа открывается меню с дополнительными настройками. Если у вас не отображаются три точки на экране, значит меню Моби-С можно открыть одной из аппаратных кнопок (одиночным или длительным нажатием) вашего мобильного устройства.

Ниже расположен выпадающий список баз данных. С одного мобильного устройства можно подключаться и работать с любым количеством баз данных.

Справа от списка баз данных круглая иконка, с пиктограммой gps-спутника, открывает диалог с информацией о состоянии GPS.

Все основные функции представлены в виде кнопок.

Режим работы с заданиями

Если агенту назначены Задания, то они отображаются в верхней части программы.

Раздел Итоги дня

Содержит обобщенную информацию по результатам деятельности агента.

  • Текущая дата установленная на мобильном устройстве.
  • Имя агента.

Вкладка Документы

Информация с разбивкой по всем оформленным на КПК документам, количество и сумма.

Вкладка Визиты

  • Визиты Запланировано - Количество назначенных агенту заданий.
  • Визиты Фактически - Количество выполненных заданий.
  • Визиты Вне плана - Количество визитов в ТТ без назначенных заданий (вне плана).
  • Документы Запланировано - Количество документов запланированных через задания.
  • Документы Фактически - Количество документов оформленных через задания.
  • Документы Вне плана - Количество документов оформленных не из заданий.

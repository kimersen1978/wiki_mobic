Настройка подключения Моби-С через внешний IP

Важно! Тестовая версия Моби-С (без лицензий) и функция Настроить мобильное устройство, работает только на порту 1234.

Прямой обмен данными между учетной системой и мобильной частью Моби-С через мобильный интернет (GPRS/3G/4G/LTE). Для настройки нужен статический IP адрес.

1. Получение статического IP адреса у вашего провайдера интернета. Статический IP обеспечит постоянный адрес вашей базы 1С в интернете и позволит производить обмен из любой точки земного шара.

Статический IP адрес выдается провайдером вашего интернета. Этот IP адрес используется в течение неограниченного промежутка времени и не может быть присвоен другому устройству. При отключении или повторном подключении компьютера к сети адрес не изменяется, как это происходит с динамическими IP-адресами.

2. Настройка порта на котором работает Моби-С. Обычно требуется проброс порта с вашего маршрутизатора, прокси сервера и пр. на компьютер где запущена обработка Моби-С. Порт необходимо открыть для протоколов TCP и UDP. Порт указан на первой вкладке модуля интеграции (по умолчанию это 1234).

Проброс порта процедура сугубо индивидуальная и зависит от конфигурации вашей сети, оборудования и установленного ПО. Для этой процедуры желательно пригласить грамотного технического специалиста. Много информации можно найти в интернете просто набрав в поисковике фразу "проброс порта".

Проверьте правильность проброса порта.

3. Настройте доступ в Интернет на КПК. В качестве теста откройте браузер и введите адрес ya.ru, загрузите страничку, в поле поиска введите любую фразу и нажмите кнопку Найти. Если отобразятся результаты поиска по введенной фразе, значит интернет на КПК работает.

4. Настройка параметров обмена на КПК. Добавьте настройки подключения по статическому IP на КПК торгового агента.

Допустим ваш статический IP 93.32.210.3, используемый в Моби-С порт 1234.

  • Перейдите на КПК Настройки ⇒ Базы данных ⇒ Ваша база".
  • Откроется диалог с настройками базы. Нажмите пункт Добавить сервер.
  • Введите название, например "Внешний".
  • Откроется диалог настроек сервера. Нажмите на поле Адрес и введите 93.32.210.3
  • Проверьте поле Порт сервера, по умолчанию стоит значение 1234, если у вас используется другой порт, то измените.
  • Закройте настройки подключения и базы данных.

5. Проверьте обмен, выполнив на КПК запрос Обновление данных ⇒ Полная загрузка.

# Требования к компьютеру для работы с последней версией платформы 1С и Windows 11

## Типовые конфигурации под разные задачи

| Компонент | Базовая (1–2 пользователя, файловый режим) | Оптимальная (клиент‑сервер, несколько баз) | Профессиональная (разработка, тяжёлые расчёты) |
|----------|---------------------------------------------|-------------------------------------------|-----------------------------------------------|
| Процессор | Intel Core i3 / AMD Ryzen 3 | Intel Core i5 / AMD Ryzen 5 | Intel Core i7 / AMD Ryzen 7 (и выше) |
| ОЗУ | $8$ ГБ DDR4 | $16$ ГБ DDR4 | $32$ ГБ+ DDR4/DDR5 |
| Накопитель | SSD $256$ ГБ (предпочтительно NVMe) | SSD $512$ ГБ (NVMe) | SSD $1$ ТБ (NVMe), опционально второй SSD |
| Видеокарта | Встроенная | Встроенная | Опционально (по задаче) |
| ОС | Windows 10/11 64‑bit | Windows 10/11 64‑bit | Windows 10/11 64‑bit |
| Сеть | Стабильное интернет‑подключение | Гигабитный LAN | Гигабитный LAN, резервные каналы |

---

## Рекомендации и выводы

* **Самый эффективный апгрейд:** замена HDD на SSD и увеличение ОЗУ до $16–32$ ГБ. Это даёт максимальный прирост производительности в 1С.
* **Для клиент‑серверного режима** критически важна стабильность сети — предпочтительно проводное подключение.
* **Надёжность системы:** качественная материнская плата и блок питания, хорошее охлаждение, регулярные резервные копии.
* **Обслуживание:**
  * Регулярные обновления платформы 1С.
  * Обслуживание базы данных согласно принятым регламентам.
  * Контроль свободного места на SSD, актуальность драйверов, предотвращение перегрева.

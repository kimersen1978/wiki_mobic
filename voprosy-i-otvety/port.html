Порт необходим для настройки обмена через GPRS. Сетевой порт это логическое соединение, через которое принимаются и отправляются данные. По умолчанию в Моби-С используется порт с номером 1234. Для того что бы шел обмен данными, необходимо настроить одинаковый порт в учетной системе и на КПК. 

Используемый порт отображается на первой вкладке модуля интеграции.

Важно! Тестовая версия Моби-С (без лицензий) и функция Настроить мобильное устройство, работает только на порту 1234.

Как проверить проброс порта?

При настройке обмена данных с использованием статического IP требуется проброс порта с вашего маршрутизатора или прокси сервера на компьютер где запущена Моби-С. Порт необходимо открыть и пробросить для протоколов TCP и UDP.

Как изменить порт в Моби-С?

В конфигураторе 1С откройте внешнюю обработку Моби-С. В общем модуле найдите и измените переменную Порт. Сохраните модуль интеграции.

Файл Интеграция Моби-С с 1С 8.epf и Интеграция Моби-С с 1С Управление торговлей 11.epf для 1С Предприятие 8.x инициализация переменной Порт находится в низу модуля, Действия ⇒ Открыть модуль объекта.

Файл Интеграция Моби-С с 1С 7.ert для 1С Предприятие 7.7 (старая версия)  переменная находиться в текстовом файл База данных\ExtForms\MobiS\ГлобальныйМодульМобиС.txt

Важно! Если вы изменили порт не забудьте изменить правила доступа и проброс порта в ваших файрволе, прокси сервере или маршрутизаторе.

 

Проверка занятости порта 1234

1.Нажмите Пуск ⇒ Выполнить, либо комбинацию клавиш Win + R.

2.Наберите команду CMD нажмите Enter либо кнопку OK В открывшемся окне консоли набрать следующее netstat -ano|findstr :1234

В ответ получите следующее сообщение:

TCP    0.0.0.0:1234           0.0.0.0:0              LISTENING       5144

UDP    0.0.0.0:1234           *:*                                    5144

Цифры 5144 это ID запущенного процесса который занимает порт.

Для того чтоб освободить порт, необходимо погасить процесс. Выполнить это можно тремя способами:

  1. Открыть Диспетчер задач. Найди номер процесса в колонке PID и завершить его работу.
  2. В консоли набрать команду taskkill /F /IM 5144
  3. Перезапустить компьютер.

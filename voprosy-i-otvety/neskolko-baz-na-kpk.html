Моби-С на КПК позволяет подключаться к любому количеству баз 1С, при этом используется всего одна лицензия Моби-С.

Важно! Во всех базах должна использоваться одна и та же лицензия Моби-С.

Устанавливаем Моби-С для каждой базы по инструкции и настраиваем обмен. На КПК тоже  создаем несколько баз и настраиваем их на свои базы 1С (код агента и настройки подключения).

Если обмен данными настроен через облачный сервис Mobi-С.Net для каждой базы 1С необходимо использовать свой КОД БАЗЫ.

Для каждой базы необходимо использовать свой ОСНОВНОЙ КАТАЛОГ.

Если обмен данными настроен через статический IP адрес и обе базы 1С находятся на одном компьютере или одном IP, необходимо, что бы каждая обработка Моби-С запускалась на своем ПОРТУ.

Добавление лицензии с онлайн активацией

1) Добавьте лицензию в обе базы 1С. Код агента может различаться, главное что бы лицензия в каждой базе вела на один КПК.

Внимание! Если на одну и туже лицензию в разных базах назначить разные КПК, будет происходить постоянная переактивация и в итоге получите сообщение Ошибка активации: Нельзя активировать один и тот-же ключ более 5 раз в сутки и больше обмен в текущем дне работать не будет.

2) Активируйте лицензию в первой базе выполнив на КПК запрос Полная загрузка. 

3) Перейдите в список ключей во второй базе 1С и нажмите кнопку Применить. Смысл в том что во вторую базу мы загрузим активацию сделанную в первой базе.

4) Выполните на КПК запрос Полная загрузка во вторую базу 1С.

Если все настроили правильно, то агент будет иметь возможность выбирать нужную базу данных на КПК из списка доступных баз на главном окне программы и при обмене не будет происходить переактивация ключа.

Если вы используете лицензии с привязкой к оборудованию, то нужно просто добавить лицензию для КПК в каждой базе.

Использование нескольких агентов на одном КПК из одной базы 1С

Если необходимо, чтобы два агента работали на одном КПК с одной базой 1С, выполните следующие шаги:

Настройка в 1С Моби-С:

  • В списке лицензий Ключи КПК добавьте лицензию дважды.
  • Для каждой из добавленных строк привяжите соответствующего агента.

Настройка на КПК:

  • Создайте две отдельные базы с разными кодами агентов.

Проверка:

  • Проверьте обмен данными в обеих базах.
  • Убедитесь, что повторной активации лицензии не требуется.

 

Использование нескольких лицензий на 1 КПК

Использовать одну лицензию на нескольких базах КПК сложно из-за активации. Она требует одновременного доступа ко всем базам, что иногда невозможно. В таких случаях можно купить и активировать лицензию для каждой базы. В приложении Моби-С нет стандартной поддержки нескольких лицензий, поэтому мы предлагаем нестандартные варианты.

 

Использование нескольких мобильных устройств

Для каждой базы требуется своё мобильное устройство, на которое устанавливается Моби-С и настраивается на одну базу. Также для каждой базы нужна своя лицензия. Это самый простой, но и самый неудобный способ.

 

Использование функции или программы для клонирования приложения

В Google Play есть много приложений, которые позволяют запускать несколько копий установленной программы. Но большинство из них не могут клонировать Моби-С, потому что для работы ему нужно много разрешений. На некоторых моделях мобильных устройств есть встроенные средства клонирования приложений.

Приложения в которых Моби-С работает

  • Clone App (не работает GPS)
  • Для телефонов Samsung можно использовать Knox. Разместить туда Моби-С и будет на КПК 2 копии.

Приложения в которых Моби-С не запустилось

  • Parallel Space
  • Multi Parallel
  • Multiple Accounts: Dual Space
  • MULTI SPACE — МУЛЬТИАККАУНТ
  • МУЛЬТИ КЛОН
  • App Cloner Premium & Add-ons
  • 2 аккаунта

 

Изменить имя пакета apk файла 

  1. Скачать с нашего сайта apk файл мобильного приложения.
  2. С помощью приложения  APK Editor Pro изменить имя пакета (инструкцию ищите в интернете). Есть аналоги этого приложения для Андроид и ПК. В результате вы получите новый файл apk.
  3. С помощью этого файла можно установить вторую версию Моби-С на одно устройство.

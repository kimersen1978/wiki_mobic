Если на КПК при вводе дробное количества не сохраняется и сбрасывается в целое число значит для этого товара запрещен ввод дробного количества. На КПК дробное количество можно ввести для товара у которого в базе 1С установлен признак "**Весовой"**. Для различных конфигураций 1С, признак "**Весовой"** может устанавливаться по-разному. После установки признака "**Весовой"** в карточке товара в 1С, на КПК необходимо выполнить запрос [**Полная загрузка**](/mobilnyj-klient-na-android/obmen-dannymi/polnaya-zagruzka).

## **Примеры установки признака Весовой для разных конфигураций 1С**

### Для конфигураций на базе УТ 10

В карточке товара установлена галка **Весовой товар**.

![](/poleznoe/faq1.png)

### Для конфигураций на базе УТ 11 

В карточке товара **единица для хранения** содержит единицу с типом измеряемой величины **Вес**.

![](/poleznoe/faq2.png)

### Для конфигураций на базе УНФ

В карточке товара установлена галка **Весовой.**

![](/help96.png)

Единица добавлена в список в настройках Моби-С, вкладка **Сервис** ⇒ кнопка **Параметры** ⇒ вкладка **Прочее.**

![](/help97.png)

### Для конфигураций 7.7

В карточке товара установлена галка **Весовой.**

![](/poleznoe/faq3.png)

Для конфигурации **Бухгалтерия 3.0 и УНФ** в Моби-С есть специальная настройка для выбора весовых единиц измерения.

В обработке Моби-С перейдите **Вкладка Сервис ⇒ Кнопа Параметры ⇒ Вкладка Прочее.**

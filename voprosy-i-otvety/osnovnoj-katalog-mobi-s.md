Основной каталог используется для хранения информации созданной в результате использования программного комплекса Моби-С, а также файлов необходимых для его работы. В каталоге хранятся: gps треки и фото сделанные агентами, лог файлы, лицензии, обновления и прочие файлы.

## **Где посмотреть и как изменить основной каталог Моби-С?**

Текущий путь к основному каталогу Моби-С можно посмотреть на [главном экране](https://wiki.mobi-c.ru/ru/integraciya-s-1s-8-ut-10) модуля интеграции Моби-С.

Для конфигураций на базе платформы 1С Предприятие 8.x путь к основному каталогу можно изменить в диалоге [**Параметры**](/integraciya-s-1s-8-ut-10/servis/parametry).

![](/poleznoe/ut10-7.png)

Для конфигураций на базе платформы 1С Предприятие 7.7 путь к основному каталогу можно изменить в файле **ГлобальныйМодульМобиС.txt**, переменная **КаталогМобиС**.

В качестве пути к каталогу необходимо указать **полный сетевой путь** к папке и предоставить на нее **полные права** для всех пользователей!

## **Что будет, если указать локальный путь?**

Локальный путь к папке имеет вид **C:\\Folder**. Используя локальный путь, при запуске на другом компьютере могут возникнуть проблемы.

-   Проблема с лицензированием. На другом компьютере каталог не доступен и лицензирование работать не будет, агенты получат сообщение **Доступ ограничен**.
-   Проблема с данными агентов, GPS треками и фотографиями. Эти файлы будут сохранены на втором компьютере и недоступны на всех остальных.
-   Проблема с просмотром отчетов и некоторыми функциями. Будут некорректно работать: отчет по маршруту агента, фото-отчет, сообщения и отчет по остаткам.

Все эти проблемы будут решены, если вы укажете полный **сетевой путь** к основному каталогу Моби-С. В качестве пути к основному каталогу Моби-С необходимо указывать полное сетевое имя каталога, чтобы пользователи с других компьютеров могли иметь к нему полный доступ. Сетевой путь имеет вид \\\\Server\\Folder. Необходимо "расшарить" каталог, получить его сетевой путь и изменить в настройках основного каталога Моби-С локальный путь на сетевой.

[Как расшарить папку без пароля в Windows 7](http://ru.d-ws.biz/articles/windows-7-share-files.shtml) [Как расшарить папку без пароля в Windows 8](http://ru.d-ws.biz/articles/windows-8-share-files.shtml)

## **Как перенести основной каталог Моби-С?**

Изменить основной каталог можно в диалоге [**Параметры**](/voprosy-i-otvety/osnovnoj-katalog-mobi-s). Если вы хотите изменить расположение основного каталога Моби-С то придерживайтесь следующего алгоритма.

1.  Закройте все запущенные обработки Моби‑С, которые настроены на текущий основной каталог.
2.  Выберите диск, на котором будет размещён новый каталог и создайте на нем пустую папку.
3.  Скопируйте в эту папку все файлы и папки из текущего основного каталога Моби-С.   
    _Важно:_ дождитесь полного завершения копирования — убедитесь, что все данные перенесены без ошибок.
4.  Запустите обработку Моби‑С. Перейдите на вкладку **Сервис**. Нажмите кнопку **Параметры**. Укажите путь к новому основному каталогу Моби‑С. Сохраните изменения.

## **Содержимое основного каталога Моби-С**

В зависимости от используемой конфигурации 1С, основной каталог Моби-С может содержать различный набор папок и файлов.

Папка \[Agents\] предназначена для хранения фотографий и выгружаемых с КПК GPS координат перемещения агентов

Содержимое папки **Agents** выглядит так:

_\[Agents\]_

 _\[00000025\] - Подпапка ТП, названием является код ТП._

         _\[Photo\] -  подпапка для выгрузки фотографий_

            _\[Clients\] - Содержит фотографии (витрины, фейсинг, документы) контрагентов выгруженные с КПК._

            _\[Goods\] - Содержит фотографии товаров выгруженные с КПК._

            _\[..\] - прочие папки для_ [_фото_](https://mobi-c.ru/help/mobi_photo.htm)_._

 _\[GPS\] - Подпапка для выгрузки GPS треков перемещения ТП._

Папка \[**ExtReps**\] предназначена для хранения файлов внешний подключаемых отчетов. Используется только для конфигураций на базе платформы 1С Предприятие 8.х. В названии отчета указывается для какой они платформы 8.1 или 8.2.

Папка \[**Forms**\] предназначена для хранения печатных и анкетных форм, файлы формата xml. Анкетные формы создаются в [визуальном редакторе](https://mobi-c.ru/help/formeditorabout.htm) анкет. [Создание и редактирование печатных форм](https://mobi-c.ru/help/mobi_printform.htm).

Папка \[**Icons**\] предназначена для хранения иконок, которые участвуют в различных отчетах Моби-С.

Папка \[**Logs**\] предназначена для хранения лога работы Моби-С (фиксация операций обмена данными и ошибки обмена), так же сюда выгружается лог работы Моби-С на КПК. Лог работы серверной части представляет собой текстовый файл имя которого состоит из приставки PS\_ и даты (пример PS\_2009-12-16.txt). Лог работы Моби-С разбит по дням. Лог работы с КПК представляет собой текстовый файл имя которого состоит из приставки ErrorLog\_ плюс код или имя ТП (пример ErrorLog\_00000037.txt). Он содержит все ошибки мобильного устройства ТП за весь период работы.

Папка \[**Updates**\] (УСТАРЕЛА) предназначена для хранения файлов автоматического обновления мобильной части (обновление программы на КПК). Подробно содержимое этой папки и процедура обновления описана в базе знаний Моби-С.

Папка \[**IrfanView**\] предназначена для хранения файлов программы для просмотра фотографий клиентов и товаров. В ней должны находится файлы  i\_view32.exe и i\_view32.ini.

Папка \[**img\_cache**\] предназначена для кэширования фотографий товаров, клиентов и заданий выгружаемых агентам на КПК, ускоряет подготовку фотографий при выгрузке на КПК агенту.

Папка \[**img\_tmp**\] предназначена для хранения фотографий товаров, клиентов и заданий отображаемых в журнале фотографий.

Папка \[**router\_log**\] предназначена для хранения логов запросов к серверу .геокодирования и доставки

**Mobi-C.chm** - справочная система по работе с программным комплексом Моби-С. Содержит информацию по всем аспектам работы Моби-С. Мобильная, серверная часть, руководство пользователя, информация для разработчиков.

[Emailer.exe](https://mobi-c.ru/help/emailer.htm) - сервисный файл, позволяет сформировать и отправить e-mail в техническую поддержку Моби-С.

**FormEditor.exe** - визуальный редактор анкетных форм. Подробная информация в справочной системе в разделе "[Редактор форм](https://mobi-c.ru/help/formeditorabout.htm)".

**mobi\_s.sqlite3** или **mobi\_s.mdb** - база для хранения выгружаемых с КПК остатков, сообщений агентам, данные по фото и проча информация.

**lickeys\_utf8.txt** - содержит данные по онлайн лицензиям Моби-С

**blacklist.bin** - необходим для правильной работы активации лицензий Моби-С

В процессе работы Моби-С может генерировать дополнительные файлы и папки.

## **Как уменьшить основной каталог Моби-С?**

1.  Удалить содержимое каталога **Logs**.
2.  Удалить из папки **Agents** папки уже не работающих агентов.

## **При настройке резервного копирования основного каталога Моби-С каталога из списка резервируемых данных можно исключить:**

_\\Agents\\<код агента>\\Photo\\Device_ – выгруженные на КПК агента фотографии, восстановятся при полной загрузке

_\\img\_cache_ – заполнится при выполнении полной загрузки/обновления данных

_\\img\_tmp_ – заполнится по мере просмотра фотографий товаров, клиентов и заданий через журнал фотографий

логи

_\\Logs_ – файлы «PS\_Дата» - логи обменов с КПК за указанную дату, если нет вопросов по работе агентов – из резервирования можно исключить, или резервировать только «за последнюю неделю» / «с момента прошлого резервирования»

_\\Logs_ – файлы «ErrorLog» - логи работы приложения Моби-С на КПК агента, резервировать не нужно, данные фиксируются в файле внешней базы данных Моби-С

_\\router\_log_ – логи запросов к серверу геокодирования и доставки, служебные файлы по которым можно восстановить обезличенную историю запросов и на получение координат и расчета доставки, отладочная информация.

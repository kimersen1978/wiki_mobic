## **Запуск обработки Моби-С с раздельными настройками Агентов (1С 8.х)**

1.  Сделать копию модуля интеграции Моби-С
2.  В обработке изменить [порт](/voprosy-i-otvety/port).
3.  Изменить название файла настроек агентов.

-   Открыть модуль интеграции в отладчике.
-   Найдите строку **РССохраненныеНастройкиИмяОбъекта = "Mobi-C".**
-   Изменяете её например на **РССохраненныеНастройкиИмяОбъекта = "Mobi-C1".**
-   Сохраните модуль интеграции.

## **Запуск обработки Моби-С с раздельными настройками Агентов (1С 7.7)**

Модуль интеграции Моби-С находиться в Каталог базы данных/ExtForms. В него входит файл **Интеграция Моби-С с 1С 7 (5.5).ert** и папка **MobiS** с дополнительными файлами. Первая копия как работала так и будет работать, её мы не трогаем.

1.  Делаем копию файла **Интеграция Моби-С с 1С 7 (5.5).ert**, назовем его **Интеграция Моби-С с 1С 7 (5.4)1235.ert**. Добавим к имени файла порт на котором будет запущен модуль интеграции. Вообще имя этого файла может быть каким угодно.
2.  Делаем копию файла **ГлобальныйМодульМобиС.txt**, который находится в папке MobiS. Назовем его **ГлобальныйМодульМобиС1235.txt**.
3.  Запускаем конфигуратор 1С и открываем в нем файл Интеграция Моби-С с 1С 7 (5.4)1235.ert. В программном модуле ищем строку **ПутьГлобальныйМодуль = "ExtForms\\MobiS\\ГлобальныйМодульМобиС.txt"**;, она находится в самом низу модуля. Изменяем имя файла глобального модуля на **ГлобальныйМодульМобиС1235.txt**. Должно получиться **ПутьГлобальныйМодуль = "ExtForms\\MobiS\\ГлобальныйМодульМобиС1235.txt";**. Там же ищем строку **ТаблицаКешаЦены = "Prices";** Изменяем имя таблицы, для каждого модуля свое. Длинна названия таблицы не может превышать 8 символов. Должно получиться ТаблицаКешаЦены = "Prices1";
4.  Открываем в текстовом редакторе файл **ГлобальныйМодульМобиС1235.txt**. Ищем строку **Порт=1234;**, она находится в самом низу. Изменяем порт на тот который нам нужен. Должно получиться **Порт=1235;**.
5.  Открываем в текстовом редакторе файл **ГлобальныйМодульМобиС1235.txt**. Ищем строку **ПараметрыАгентов = "ПараметрыАгентов.txt";**. Изменяем имя файла настроек. Должно получиться **ПараметрыАгентов = "ПараметрыАгентов1235.txt"**;.

Запускать каждую копию модуля интеграции Моби-С необходимо под своим пользователем 1С. Во всех модулях открытых таким способом список и настройки агентов будут разные.

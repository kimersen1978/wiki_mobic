# **Как добавить нового агента Моби-С?**

В видеоролике показано как добавить нового агента Моби-С. У вас должен быть уже настроен обмен данными и приобретены лицензии. Добавление агента Моби-С будет показано для конфигурации "**Управление торговлей 11**". Моби-С поддерживает [более 50 различных конфигураций](https://mobi-c.ru/1cconfiguration.html) и в них интерфейс может отличаться, но суть остается та же.

## **Краткий алгоритм добавления нового агента**

1.  Добавить нового агента Моби-С в [список агентов](https://wiki.mobi-c.ru/ru/integraciya-s-1s-8-ut-10/agenty).
2.  Привязать нового агента к [лицензии](/integraciya-s-1s-8-ut-10/servis/klyuchi-kpk). Если нет свободной лицензии смотрите видео по покупке лицензии.
3.  Установить Моби-С на мобильное устройство из магазина приложений Google Play или с нашего сайта.
4.  Загрузить настройки на КПК с помощью функции "[**Получить настройки**](/mobilnyj-klient-na-android/nastrojki/poluchit-nastrojki)" или в ручную.

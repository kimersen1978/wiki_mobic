Внешняя обработка (в дальнейшем модуль интеграции) предназначена для настройки обмена между мобильной частью  Моби-С и учетной системой на базе платформы 1С:Предприятие, настройки параметров выгрузки и работы торговых агентов, просмотра отчетов. Список поддерживаемых конфигураций 1С

Важно! Для первого запуска обработки Моби-С необходимо чтобы у пользователя 1С были полные права. Последующий запуск Моби-С в режиме обмена с КПК рекомендуется выполнять от имени пользователя 1С с полными правами. Если необходимо ограничить права пользователя, то воспользуйтесь статьей Права пользователя 1С необходимые для работы Моби-С.

Подключение модуля интеграции не требует каких-либо доработок типовой конфигурации. Модуль интеграции выполняет обмен данными с КПК. Обмен с КПК возможен только тогда, когда открыто окно данной обработки.

Модуль интеграции Моби-С состоит из двух частей: Главное окно и Верхнее меню.

Главное окно, вкладка Сервер обмена

Если Режим обмена с КПК - Включен, значит обработка используется в качестве сервера обмена 1С с мобильными устройствами.

Если Режим обмена с КПК - Выключен, в этом режиме работают все функции Моби-С, кроме обмена данными. Далее указана причина отключения режима обмена.

Редакция - версия модуля интеграции Моби-С. Актуальную версию модуля интеграции можно получить на нашем сайте в разделе Скачать.

Порт - используемый при обмене через IP, порт.

Статус облачного сервера Mobi-C.Net - показывает текущее состояние Mobi-C.Net. Настройка облачного сервера Mobi-C.net.

PocketServer.dll - дата сборки серверной компоненты.

Основной каталог - путь к основному каталогу Моби-С. Настройка основного каталога Моби-С. Что хранится в основном каталоге Моби-С?

Галка Режим отладки - Включает в модуле интеграции расширенное логирование.

Важно! Включение режима отладки может значительно замедлить обмен данными. Ставьте галку только если вам нужна информация для поиска ошибок в работе Моби-С.

Значение Коррекция даты клиента показывает на сколько дней текущая дата отличается от даты установленной на компьютере. Если на вашем компьютере дата отличается от текущей вам необходимо настроить суточную коррекцию в Настройках. Эта настройка необходима для корректной работы с GPS данными и фотографиями

Верхнее меню

Используя верхнее меню, вы получите доступ к основным функциям модуля интеграции.

Изменение модуля интеграции

Модуль интеграции полностью открыт для изменения и доработки. Программист 1С может добавить или изменить алгоритмы выгрузки/загрузки данных, параметры создания документов, выгружаемые на КПК отчеты и многое другое. Для того, чтобы использовать модуль интеграции с конфигурацией отличной от поддерживаемых, необходимо адаптировать его под новую конфигурацию с учетом их различия и в соответствии с протоколом обмена.
